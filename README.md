## 理论体系

**水系统控制论 (Cybernetics of Hydro Systems, CHS)** 是面向水网自主运行的工程控制理论框架，包含：

- **八原理**: 传递函数化、可控可观性、分层分布式、安全包络、在环验证、认知增强、人机共融、全生命周期自主演进
- **WNAL L0-L5**: 水网自主等级分级体系
- **HydroOS**: 水网操作系统架构
- **MBD**: 基于模型的设计验证一体化框架
