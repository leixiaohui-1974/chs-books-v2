# 水系统现代控制

CHS教材体系补充教材 — 控制方法在水系统中的深度工程实现

## 定位

本书不是CHS理论教材（那是T1/T2a/T2b的角色），而是**控制方法在水系统中的深度工程实现手册**。理论框架引用CHS体系，在具体方法的水利案例、数学推导、Python代码实现上做到5本代表作无法覆盖的深度。
