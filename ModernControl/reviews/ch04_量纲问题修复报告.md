## 1. 问题描述

**位置**: ch04_final.md 行 483-487

**原公式**（量纲错误）:
```
$$
\tau_m = \frac{T_p}{2}\left(1 - \frac{2Q_0}{g A_0 c}\right)
$$
```

**量纲分析**:
- $\frac{2Q_0}{g A_0 c} = \frac{[\text{m}^3/\text{s}]}{[\text{m}/\text{s}^2] \cdot [\text{m}^2] \cdot [\text{m}/\text{s}]}$
- $= \frac{[\text{m}^3/\text{s}]}{[\text{m}^4/\text{s}^2]} = \frac{[\text{s}]}{[\text{m}]}$ ❌

**问题**: 括号内的项不是无量纲量，导致整个公式量纲不一致。

---

## 2. 修正方案

**修正公式**:
```
$$
\tau_m = \frac{T_p}{2}\left(1 - \frac{2V_0}{c}\right)
$$
```

其中 $V_0 = Q_0 / A_0$ 为正常流速 [m/s]。

**量纲验证**:
- $\frac{2V_0}{c} = \frac{[\text{m}/\text{s}]}{[\text{m}/\text{s}]} = $ 无量纲 ✓
- $\tau_m = [T] \cdot (1 - \text{无量纲}) = [T]$ ✓

**物理意义**:
- $V_0/c$ 为 Froude 数的平方根项，表征流速与波速的比值
- 当 $V_0 \ll c$ 时（低流速），$\tau_m \approx T_p/2$
- 该修正与 Litrico & Fromion (2009) 原文一致

---

## 3. 案例数值验证

**表 4-2 中的参数**:
- $Q_0 = 60$ m³/s
- $V_0 = 0.85$ m/s
- $c = 5.60$ m/s
- $\tau_m = 1260$ s（表中给出）

**验证计算**:
```
2V_0/c = 2 × 0.85 / 5.60 = 0.3036

如果 τ_m = 1260 s，反推:
T_p = 2τ_m / (1 - 2V_0/c)
   = 2 × 1260 / (1 - 0.3036)
   = 2520 / 0.6964
   = 3619 s ≈ 60 分钟
```

**结论**: 表中数值 $\tau_m = 1260$ s 在修正公式下合理（约为 $T_p/2$ 的 70%）。
