### P0-2 【数学严谨性】弹性水击模型推导不完整
**来源**：Codex + Gemini
**位置**：9.2.1节第96-109行
**问题**：
- 公式 G_w(s) = (k_q / T_w s) · tanh(√(T_w s)) 缺少推导来源
- tanh(√s) 的物理意义未解释清楚（水击波传播和反射）
- "半积分特性" 1/√s 表述误导（不是标准环节）
- 低频/高频近似的适用条件未明确

**修复方案**：
1. 补充推导来源（Wylie & Streeter 1993）
2. 明确 tanh(√s) 来自双曲偏微分方程的频域解
3. 删除"半积分特性"表述，改为"频域近似"
4. 补充适用频率范围（ω < πa/L）
5. 补充过渡段落（200字）：刚性→弹性的物理机制
6. 补充示意图：刚性 vs 弹性水击的波形对比

### P0-3 【工程参数】9.7.1案例参数不自洽
**来源**：三引擎共同指出
**位置**：第656-686行
**问题**：
- 给定 T_w=2.5s, L=200m, H_0=80m, 300MW
- 缺少额定流量 Q_0、管道直径 D
- 传递函数增益 1.2 单位不明
- 工程师无法验证功率匹配性，无法复现系统

**修复方案**：
1. 补充额定流量：Q_0 = 400 m³/s
2. 补充管道直径：D = 7.2 m
3. 验证功率：P = ρgQHη = 1000×9.8×400×80×0.9 ≈ 282 MW（接近300MW）
4. 补充效率：η = 0.9
5. 补充水轮机类型：混流式（Francis）
6. 明确传递函数增益单位（rad/s per unit）或标注"参数已归一化"

### P0-4 【时滞类型】9.7.2案例时滞物理意义不清
**来源**：Codex + Claude
**位置**：第687-768行
**问题**：
- 纯时滞 τ=600s 的物理意义未解释
- 未明确是质量输运时滞（L/v）还是压力波时滞（L/a）
- 工程师可能误用压力波时滞设计控制器
- 50km距离，v=1.4m/s → τ=600s（质量输运），但 a=1200m/s → τ_wave=42s（压力波）

**修复方案**：
1. 明确时滞类型：质量输运时滞（advection delay）
2. 补充计算过程：τ = L/v = 50000/1.4 ≈ 600s
3. 解释：长距离输水系统，流量变化主要受质量输运影响
4. 补充：压力波时滞 τ_wave ≈ 42s << τ_advection = 600s，可忽略
5. 补充工程背景：管道直径、流速、泵站配置
