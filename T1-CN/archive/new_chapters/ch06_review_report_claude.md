### 建议1：补充直觉类比（优先级P1）

**问题描述**：
- §6.2.4（IDZ模型）的零点概念较抽象，研究生可能难以理解"回水效应"的物理意义
- §6.4（Muskingum-IDZ对偶性）的数学推导较长，缺少直觉解释

**建议方案**：

**位置1：§6.2.4补充类比**
```markdown
**直觉类比**：回水效应就像"多米诺骨牌的反向传播"。

想象一排多米诺骨牌：
- 正向传播（时滞τd）：推倒第一张牌，波动向后传播
- 反向传播（回水τm）：最后一张牌被挡住，"堵塞"信号向前传播

在明渠中：
- 上游开闸 → 水波向下游传播（τd）
- 下游闸门关闭 → 水位上升，回水波向上游传播（τm）

零点(1+τm s)就是这个"反向传播"的数学表达。
```

**位置2：§6.4.1补充对偶性直觉**
```markdown
**直觉理解**：Muskingum-IDZ对偶性就像"牛顿力学与相对论的关系"。

- 牛顿力学：低速近似，简单实用（Muskingum）
- 相对论：高速精确，复杂完整（IDZ）
- 当速度v<<c时，相对论退化为牛顿力学

类似地：
- 当频率ω→0（慢变化）时，IDZ退化为Muskingum
- 这不是巧合，而是数学上的必然联系
```

**优先级**：P1（重要但非紧急）
