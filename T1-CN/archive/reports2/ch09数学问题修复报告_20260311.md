### ✅ 问题1: Nyquist判据表述风险（行387-396）

**原问题**:
- 表述"逆时针包围点(-1, 0)的圈数等于开环传递函数在右半平面的极点数"容易导致结论反号

**修复内容**:
- 改为标准形式：$Z = N + P$（Z为闭环右半平面极点数，N为顺时针包围圈数，P为开环右半平面极点数）
- 明确稳定条件：$Z = 0 \Rightarrow N = -P$
- 单列P=0时的特殊情况："Nyquist曲线不包围点(-1, 0)"

**修复后的表述**:
```
设开环传递函数G(s)H(s)在右半平面有P个极点，Nyquist曲线顺时针包围点(-1, 0)的圈数为N，
则闭环系统在右半平面的极点数为: Z = N + P

闭环系统稳定的充要条件是Z = 0，即: N = -P

说明:
- 若P = 0（开环稳定），则N = 0，即Nyquist曲线不包围点(-1, 0)
- 若P > 0（开环不稳定），则N = -P，即Nyquist曲线需逆时针包围点(-1, 0)共P圈
```

### ✅ 问题2: S区模型分母不严谨（行599-605）

**原问题**:
- 分母写成 `2Hs - |e_mn|`，遗漏阻尼项D
- 使用绝对值符号不准确

**修复内容**:
- 改为 `2Hs + D - e_mn`
- 明确不稳定条件：`e_mn > D`（而非使用绝对值）
- 补充物理解释：当e_mn > D时，分母中出现负阻尼项

**修复后的表述**:
```
在S区，水轮机的转速-力矩特性系数e_mn > 0（正反馈），系统传递函数变为:

G(s) = K / [(T_w s + 1)(2H s + D - e_mn)]

当e_mn > D时，分母中出现负阻尼项，导致右半平面极点，系统不稳定。

不稳定条件: e_mn > D ⇒ 系统存在右半平面极点
```

### ✅ 问题4: 增量变量不一致（行40-58）

**原问题**:
- 方程里混用q与Δy
- 拉氏变换直接写Q/Y，未说明增量变量关系

**修复内容**:
- 明确声明"工作点线性化"和"增量变量定义"
- 统一使用Δq, Δy表示增量
- 补充"零初值条件"说明
- 明确Q(s) = L{Δq}, Y(s) = L{Δy}

**修复后的表述**:
```
线性化假设: 在工作点(q_0, y_0)附近进行小扰动分析，定义增量变量:
Δq = q - q_0,  Δy = y - y_0

线性化后的动态方程为:
T_w d(Δq)/dt + Δq = k_q Δy

传递函数推导: 对上式进行拉普拉斯变换（零初值条件）:
T_w s Q(s) + Q(s) = k_q Y(s)

得到传递函数:
G_w(s) = Q(s)/Y(s) = k_q/(T_w s + 1)

其中Q(s) = L{Δq}，Y(s) = L{Δy}为增量变量的拉普拉斯变换。
```
