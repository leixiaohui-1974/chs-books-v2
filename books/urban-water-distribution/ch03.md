# 第3章 水质衰减模型（余氯）

## 导读

饮用水的输配安全是现代公共卫生的基石。自20世纪初液氯消毒技术广泛应用于城市供水系统以来，水型肠道传染病得到了有效遏制。上一章解决了管网中“水往哪里流、压力有多大”的水力学基础问题，本章将进一步深入回答“水质在输配过程中如何变化”的核心科学问题。饮用水在出厂前通常会经过系统的加氯消毒处理，然而，带有初始余氯的净水在管网长距离输配、多级调蓄以及用户端停留的复杂过程中，余氯会因各种化学反应和管壁微生物作用而持续衰减。这种衰减带来了一个严峻的水质管理两难困境：管网末梢余氯浓度过低，将无法有效抑制致病微生物和管网生物膜的再生长，引发水源性疾病二次爆发风险；反之，若为了保证末梢余氯而在水厂过量加氯，较高的初始游离氯会与水中的天然有机物（NOM）发生复杂的卤代反应，生成三卤甲烷（THMs）、卤乙酸（HAAs）等具有致癌、致畸、致突变风险的消毒副产物，严重威胁人体长期健康。因此，在错综复杂的城市地下管网中，准确预测并控制余氯的时空分布，寻找微生物安全与化学安全之间的“黄金平衡点”，是保障供水水质的关键技术。本章将从微观的反应动力学出发，建立管网水质传输与反应的宏观数学模型，详细推导余氯在单根管段内的一维对流-反应方程及其稳态解析解。在此基础上，系统介绍多组分反应动力学模型和综合表征管网滞水状态的水龄模型，并通过全周期动态仿真，深度剖析余氯在管网不同工况下的衰减空间分布规律与时间演变特征。本章构建的水质建模理论体系，将为后续第4章的管网漏损水质异常检测、第7章基于数字孪生的水质状态实时估计与预测提供坚实的数学与物理基础。

## 3.1 余氯衰减动力学

余氯在给水管网中的消耗是一个多相、多组分耦合的复杂物理化学过程。根据反应发生的空间位置，余氯衰减机理可清晰地划分为两类：发生于流动水体内部的化学衰减（即体相反应）以及发生于管壁界面处的接触反应（即壁面反应）。

### 3.1.1 一级反应模型

体相反应主要源于水体中残留的还原性无机物（如二价铁、二价锰、氨氮）以及溶解性天然有机物与次氯酸（HOCl）或次氯酸根（OCl⁻）之间的氧化还原反应。在工程应用中，最简单且应用最为广泛的是一级反应动力学模型。该模型基于碰撞理论的简化假设，认为在给定水质背景下，还原性物质的浓度相对过剩或保持动态稳定，因而余氯的衰减速率仅与当前时刻的余氯浓度自身成正比。其微分控制方程如下：

$$
\frac{\mathrm{d}C}{\mathrm{d}t} = -k_b \, C \tag{3.1}
$$

其中，$C$ 为任意时刻水体中的余氯浓度（mg/L），$k_b$ 为体相衰减系数（h$^{-1}$）。$k_b$ 的典型值通常分布在 $0.2 \sim 1.5$ h$^{-1}$ 的区间内，其具体大小高度依赖于水体的物理化学性质，包括原水有机物类型、pH值以及浊度等。对式(3.1)进行分离变量并积分，应用初始条件 $t=0$ 时 $C=C_0$，即可推导得到该模型的时间域解析解，呈现典型的指数衰减规律：

$$
C(t) = C_0 \, e^{-k_b t} \tag{3.2}
$$

这一指数衰减公式清晰地表明，在管网水力停留时间增加的过程中，余氯并非匀速下降，而是在初始阶段衰减较快，随后逐渐变缓。除了水质组分，水温是驱动体相反应速率波动的核心热力学参数。反应速率与温度的非线性依赖关系可通过经典物理化学中的Arrhenius方程进行宏观描述：

$$
k_b(T) = k_b(T_0) \cdot \theta^{(T - T_0)} \tag{3.3}
$$

其中，$T$ 为实际运行水温（°C），$T_0$ 为实验室测定的参考温度（国际标准通常取20°C），$\theta$ 为由反应活化能决定的温度校正系数（经验值约在1.03~1.07之间）。从分子动力学角度分析，水温升高不仅增加了反应分子的热运动动能，更提高了发生有效碰撞的活化分子百分数。计算可知，水温每升高10°C，体相衰减系数 $k_b$ 将呈现几何级数增长，约增大1.3~2.0倍。夏秋交替等高温季节，管网内余氯衰减曲线显著变陡，水厂需采取提高出厂水氯量或启动中途加氯等调度手段，这是供水企业日常水质管理中必须重点应对的季节性挑战。

### 3.1.2 壁面反应模型

相较于体相反应，管壁处的余氯消耗机理更为繁杂。管壁不仅仅是一个物理边界，更是一个高度活跃的生化反应器。其消耗机制主要包括两部分：一是附着于管壁的生物膜（Biofilm）及其分泌的胞外聚合物（EPS）的新陈代谢与氧化消耗；二是金属管道（如灰口铸铁管）内壁腐蚀结节、铁锈等腐蚀产物对氧化剂的强烈还原反应。

在EPANET等主流管网水质模型中，采用壁面反应系数 $k_w$（m/h）来定量表征这一非均相反应过程。由于壁面反应发生在固液交界面，主体水流中的余氯分子必须首先通过层流底层和过渡区，克服传质阻力扩散至管壁表面，才能参与反应。因此，壁面衰减速率受制于化学反应动力学和流体动力学双重控制。考虑无限长圆管内的径向稳态传质平衡，主体水向管壁的质量通量必须等于管壁表面的反应消耗量。基于此质量守恒原理，壁面反应对管段截面平均余氯浓度的等效衰减贡献率 $r_w$ 可推导为：

$$
r_w = \frac{2 k_w k_f}{R(k_w + k_f)} \, C \tag{3.4}
$$

其中，$R$ 为管道内半径（m）；$k_f$ 为由流体湍流程度决定的径向传质系数（m/h）。$k_f$ 的物理意义在于表征水流对边界层的冲刷和更新能力，通常依赖于无量纲数Sherwood经验公式确定：

$$
k_f = \frac{Sh \cdot D_m}{D} \tag{3.5}
$$

式中，$D$ 为管道直径；$Sh$ 为Sherwood数，它代表了对流传质与分子扩散传质的比率（层流状态下理论解为定值 $Sh = 3.65$，湍流状态下多采用经验关联式 $Sh = 0.0149 Re^{0.88} Sc^{0.33}$）；$D_m$ 为余氯在水中的分子扩散系数（水温20°C时约为 $1.2 \times 10^{-9}$ m$^2$/s）；$Sc = \nu / D_m$ 为衡量动量扩散与质量扩散相对大小的Schmidt数（$\nu$ 为水的运动粘滞系数）。

将水体内部和边界界面的双重效应叠加，管段内余氯的总表现衰减系数可表达为：

$$
k_{\text{total}} = k_b + \frac{2 k_w k_f}{R(k_w + k_f)} \tag{3.6}
$$

该公式深刻揭示了管径对水质的影响：在相同的管材和水力条件下，管径 $R$ 越小，比表面积（表面积与体积之比）越大，壁面反应项占主导地位。实地调研表明，旧管道由于内壁粗糙度增加、结垢严重且生物膜厚度增加，$k_w$ 值通常比新出厂管道高出2~5倍。对于服役期超过30年的无内衬铸铁老管，$k_w$ 可高达0.5~1.0 m/h；相比之下，表面光滑且化学性质稳定的新型高密度聚乙烯（HDPE）管材，$k_w$ 仅维持在0.01~0.05 m/h水平。管材类型和服役年限是主导壁面反应速率的决定性因素，也是导致城市老城区分支管网末梢余氯长期偏低的核心物理原因。

### 3.1.3 多组分反应模型

尽管一级模型在多数常规工况下表现出较好的工程实用性，但其内在假设水体中仅存在单一活性的消耗物质，这在描述长周期衰减时存在系统性偏差。实际上，天然有机物（NOM）是由不同分子量、不同官能团组成的复杂混合物。为了更精确地捕捉余氯消耗的非线性特征，Clark等人在1998年提出了更为严密的二组分反应动力学模型。该模型创造性地将NOM划分为两类：反应活性极高的快速组分和活性较低的慢速组分。余氯与这两种组分的反应构成了并行的二级反应体系：

$$
\frac{\mathrm{d}C}{\mathrm{d}t} = -k_1 \, C \, C_{\text{fast}} - k_2 \, C \, C_{\text{slow}} \tag{3.7}
$$

$$
\frac{\mathrm{d}C_{\text{fast}}}{\mathrm{d}t} = -k_1 \, C \, C_{\text{fast}} \tag{3.8}
$$

$$
\frac{\mathrm{d}C_{\text{slow}}}{\mathrm{d}t} = -k_2 \, C \, C_{\text{slow}} \tag{3.9}
$$

其中，$k_1$ 和 $k_2$ 分别表征快速和慢速反应组分的二级反应速率常数（L/(mg·h)）；$C_{\text{fast}}$ 和 $C_{\text{slow}}$ 分别代表这两种虚拟有机物组分在水中的当量浓度。物理化学实验测定表明，快速组分主导了加氯初期的剧烈消耗，其半衰期通常仅在0.5~2小时之间；而慢速组分则决定了管网远端的持久消耗，半衰期长达10~50小时。

这一耦合非线性微分方程组完美诠释了余氯衰减的双阶段动力学特征：水厂出厂后的最初数小时内，曲线陡峭下降，此时水中高活性物质被迅速氧化（快速组分主导）；随着高活性物质消耗殆尽，反应进入由慢速组分控制的平台期，衰减速率大幅放缓。在大型跨区域长距离引调水工程或区域性供水干线中，水流在单一管道内的停留时间动辄超过10甚至20小时，若仍采用单一的一级模型，往往会严重高估末端管网的余氯消耗量。引入二组分模型，能够更客观、精准地拟合全时间域的余氯动态演化轨迹。

### 3.1.4 消毒副产物生成

在探讨余氯衰减的同时，必须将视角延伸至其伴生的化学毒理风险。余氯作为强氧化剂，在与水中腐殖酸、富里酸等大分子有机物发生反应时，不仅导致有效氯浓度的降低，更会不可逆地生成以三卤甲烷（THMs）为代表的含卤素消毒副产物（DBPs）。这类微量有机污染物具有较强的致癌潜力。

THMs的生成动力学曲线通常呈现渐近增长特征，可用基于传质和反应限制的平行一级反应生成模型加以宏观描述：

$$
C_{\text{THM}}(t) = C_{\text{THM},\infty} \left(1 - e^{-k_{\text{THM}} t} \right) \tag{3.10}
$$

其中，$C_{\text{THM},\infty}$ 定义了在余氯和反应前体物均充足的极限条件下，THMs可能达到的最终生成潜能极大值（mg/L）；$k_{\text{THM}}$ 为表观生成速率常数。《生活饮用水卫生标准》（GB 5749-2022）明确规定，出厂水及管网中总THMs的浓度限值不得超过0.06 mg/L。这就要求供水系统在进行消毒策略顶层设计时，必须在二维约束空间内求解最优控制方案：既要满足管网最远端不低于0.05 mg/L的余氯微生物安全下限，又要确保全网任意节点最高水龄处的消毒副产物不突破化学安全上限。

## 3.2 管段内水质传输方程

### 3.2.1 一维对流-反应方程

在确定了余氯的局部衰减动力学之后，需将其置于流体力学框架中进行空间拓展。对于单根输水管段，取微元控制体进行质量守恒分析：在假定管内水流经过充分发展、径向混合均匀的前提下，余氯浓度的时空演变严格遵循一维对流-反应偏微分方程。考虑到城市供水管网绝大部分处于充分发展的湍流状态（雷诺数 $Re > 4000$），水流的宏观平流输运作用占据绝对优势。通过无量纲Peclet数量级分析可知，在管网几百米至几公里的空间尺度上，纵向弥散引起的质量传递相比对流项微乎其微。工程上通常进行合理的数学简化，舍弃二阶弥散项，得到标准的一维双曲型守恒方程：

$$
\frac{\partial C}{\partial t} + u \frac{\partial C}{\partial x} = -k_{\text{total}} \, C \tag{3.11}
$$

式中，$u$ 为管段截面平均流速（m/s）；$x$ 为沿管段中心轴线方向的距离坐标（m）。对于一条长度设定为 $L$、且内部流速在一定时段内维持恒定的管段，令方程左侧的时间偏导数项 $\partial C / \partial t = 0$，对空间坐标 $x$ 积分可得其稳态解析解：

$$
C_{\text{out}} = C_{\text{in}} \, e^{-k_{\text{total}} \, \frac{L}{u}} = C_{\text{in}} \, e^{-k_{\text{total}} \, \tau} \tag{3.12}
$$

其中，$\tau = L / u$ 定义为管段的本征水力停留时间（s）。该解析解直观地揭示了沿程水质变化的内在物理机制：管段出口余氯浓度 $C_{\text{out}}$ 是入口浓度 $C_{\text{in}}$ 与负指数衰减项的乘积，而指数项的核心自变量是综合反映反应速率与水力输运时间的无量纲 Damköhler数（定义为 $Da = k_{\text{total}} \tau$）。$Da$ 数度量了化学反应特征时间与流体输运特征时间的比例关系。当 $Da > 1$ 时，意味着在水流通过该管段的时间内，该管段的余氯衰减率将超过63.2%，此类管段属于管网中的水质阻滞高危段。

### 3.2.2 节点完全混合模型

在管网的交叉节点处，来自上游不同分支管段的水流发生物理汇合。EPANET采用了理想的“节点完全混合”假设，即认为在节点极小的控制体积内，水流的紊动混合是瞬时完成的，且不发生化学反应。基于质量守恒定律，混合后的节点总浓度可表达为流量加权平均值：

$$
C_{\text{node}} = \frac{\sum_{j \in J^{+}} Q_j \, C_j^{\text{out}} + Q_{\text{ext}} \, C_{\text{ext}}}{\sum_{j \in J^{+}} Q_j + Q_{\text{ext}}} \tag{3.13}
$$

其中，$J^{+}$ 表示所有水流方向指向该节点的上游管段集合；$Q_j$ 为第 $j$ 条入流管段的体积流量；$C_j^{\text{out}}$ 为该管段计算至末端的出口浓度；$Q_{\text{ext}}$ 和 $C_{\text{ext}}$ 代表外部水源（如加氯泵站）注入节点的流量与质量浓度。这一混合模型将上一级管段的出口状态重置为下游管段的入口边界条件，从而推动全网连续计算。

完全混合假设在标准的T型三通等管径相近的节点处基本符合流体力学事实（混合长度约为管径的3~5倍）。但在大型清水池、高位水塔等大截面调蓄设施中，水体内部极易产生短流或分层现象。为此，必须引入针对性的反应器模型，如推流式模型、先进先出模型以及用于模拟死水区的两室混合模型，以还原真实设施内的水力学特征。

### 3.2.3 动态水质模拟的数值方法

城市供水系统时刻面临着用户需水量波动的动态挑战。此时，水质方程(3.11)的非定常项不可忽略，必须依赖数值方法求解。EPANET采用了一种兼顾效率与精度的离散体积法（Discrete Volume Element Method, DVEM）。

其核心思想是将整个管网中每条包含水流的管段，根据当前流速划分为若干个不连续的、内部浓度均匀的“体积段”。在水质计算推进的每一个极短时步内，这些体积段沿管道发生刚性平移，同时在内部独立计算衰减反应。当体积段的前端到达末尾溢出时，在下游节点处利用式(3.13)进行瞬时均匀混合，混合后的新水体再重新打包送入下游管段。

DVEM算法的精度与收敛性严格受制于水质计算时步 $\Delta t_q$ 的选取，必须满足数值分析中的Courant-Friedrichs-Lewy（CFL）条件：

$$
\Delta t_q \leq \frac{\Delta x}{u_{\max}} \tag{3.14}
$$

其中，$\Delta x$ 为算法允许的最小体积段长度界限；$u_{\max}$ 为全网在当前水力时间步长内出现的最高水流速度。在工程实操中，水质时步通常设定在1~5分钟，远比水力模型中常用的15分钟至1小时的宏观时步要密集，从而在微观时间颗粒度上保证了对流项离散的计算精度。

## 3.3 水龄与管网滞水分析

在管网水质评估体系中，水处理工程领域引入了“水龄”（Water Age）这一关键概念。水龄定义为特定水质点从离开水源节点进入管网开始算起，抵达某一指定节点所累积经历的时间。其一维传输方程表示为：

$$
\frac{\partial T_a}{\partial t} + u \frac{\partial T_a}{\partial x} = 1 \tag{3.15}
$$

其中，$T_a$ 表示流经位置 $x$ 处的水龄（h）。节点处水龄采用流量加权平均计算。水龄不仅是水流滞留时间的标尺，更是评估各类水质衰变程度的核心代理指标：水龄越长的区域，意味着水体自我更新能力越弱，各类水质劣化的风险成倍增加。

管网中的高水龄“滞水区”通常集中在以下三类结构瓶颈处：
(a) **末梢死端管段**：常见于城市边缘的独立居民区，管网呈树状结构，水流模式为“进出同口”，夜间无用水时完全停滞。
(b) **过度设计的调蓄水池**：当水池设计容积远超出日均用水负荷时，大量净水在池内长时间滞留。
(c) **环网内部的低压差死区**：在成环管网的特定区段，两侧水压接近平衡，流速长年徘徊在极低水平，形成隐蔽的“准死水区”。

针对高水龄顽疾，现代水司通常采取基于单向冲洗（UDF）的定期排空计划，强制提升末端置换率；或通过智能调度微调环网关键阀门开度，人为打破压力平衡，重塑流速场分布。

## 3.4 仿真设置

为了具象化展示余氯的演变动态，本节基于一个包含25个节点、34条管段的典型管网拓扑进行24小时全周期动态仿真。水源节点出厂水保持恒定的初始余氯浓度 $C_0 = 1.0$ mg/L，体相衰减系数统一取值为 $k_b = 0.5$ h$^{-1}$。

## 3.5 结果分析

仿真计算结果从空间分布和时间演化双维度呈现出清晰的规律。空间维度上，日用水量加权平均视角下，距离水源1 km半径内的核心节点群余氯充沛，维持在 $0.65 \sim 0.85$ mg/L；延伸至距水源2~3 km的中游区域，浓度滑落至 $0.35 \sim 0.55$ mg/L；当水流到达距水源4 km以上的末梢节点时，余氯降至 $0.15 \sim 0.30$ mg/L。在工况2（实际管龄分布）场景下，老城区末端的节点J22和J25日均余氯已跌破0.10 mg/L，在特定时段逼近GB 5749-2022规定的末梢余氯下限极值（0.05 mg/L）。

时间维度上，在夜间用水深度低谷时段（凌晨2:00-5:00），受管网需水量锐减影响，干管与支管流速普遍骤降，导致水体在管道内的滞留时间被拉长。数据统计揭示，此时段内各末梢节点的瞬时余氯浓度较白天高峰期大幅下降，相对降幅达20%~30%。这一趋势在凌晨3:00达到了极限组合：最远端节点J25瞬时余氯探底至微弱的0.08 mg/L。这种“夜间低氯”现象是引发夜间管壁生物膜脱落及细菌总数超标的高危因素。

比对三组独立工况的数据，当管壁特性从全网新管恶化至全网老管时，管网最远端节点的24小时平均余氯浓度发生了断崖式下跌，跌幅高达68.4%。具体观测数据为：工况1（新管）0.38 mg/L，工况2（实际混合管龄）0.22 mg/L，工况3（老管）0.12 mg/L。定量数据证实，旧管道内壁高活性的腐蚀层是吞噬管网末梢余氯的关键黑洞。

对全网节点水龄参数的统计表明，仿真周期内管网总需水量加权平均水龄为3.2小时，最大极端水龄出现在末梢死端处，高达12.5小时。频数直方图显示，水龄超过8小时的节点占据总数的15%，超过12小时的滞水节点占比达4%。同步时刻的水龄值与余氯浓度值的皮尔逊相关性计算结果显示，两者负相关系数达到 $-0.93$，完美印证了水龄作为早期水质预警核心指标的科学性。在实际管理中，将水龄超8小时区域划定为“预警区”，超12小时区域划定为“高危区”，为管网巡检提供了空间指引。

## 3.6 工程启示

将水质动力学模型转化为生产力，对智慧供水企业的运营管理具有极高的指导价值，具体体现在以下实际案例场景中：

**二次加氯站选址与动态优化**。对于大规模管网，单纯依靠水厂源头提升出厂水余氯极易导致近端用户DBPs超标。深圳市某主力水厂在向偏远区域输水时，管网模型发现末端余氯偏低。技术团队利用模型进行遗传算法寻优，最终在距主城区15公里处的某关键增压泵站内增设智能补氯系统。模型给出的选址原则是将加氯点前移至余氯刚降至0.15 mg/L的上游咽喉处，以较小的投加量，使下游广阔区域的余氯平稳拉升至0.3~0.5 mg/L，成功实现了微生物抑制与副产物控制的双赢。

**科学制定管网冲洗与维护计划**。传统的管道冲洗盲目性大、水资源浪费严重。北美某市供水局引入基于模型的单向冲洗（UDF）系统，系统自动识别出高水龄死端，并计算出最佳的阀门关停次序，确保冲洗时局部流速瞬间突破清洗临界值。实际运行数据显示，智能冲洗策略不仅使末端平均余氯回升0.2 mg/L，更将年度冲洗耗水量大幅削减了40%，创造了显著的环境与经济效益。

**水质突发污染事件的应急响应**。当遭遇意外化学物质侵入管网的极端事件时，动态水质模型如同管网的“神经中枢”。在某市抗供水突发污染演练中，水力水质模型在接报后5分钟内推演了污染羽流在地下管网中的扩散路径和到达各小区的精确时间，直接指挥中心在污染源下游精准关闭3个关键隔离阀门，有效避免了污染面的不可控蔓延。

**在线水质传感器网络的拓扑布点**。面对庞大的地下管网，供水企业不可能全节点布设昂贵的监测仪。采用国际前沿的TEVA-SPOT传感器布局优化算法针对某500节点区域进行测算，结果表明，若仅依靠人工沿主干道布设10台设备，预警覆盖面仅为40%；而依托模型将传感器配置在高水龄汇聚点和拓扑“枢纽交叉点”，仅需6个监测点即可灵敏捕捉全网80%区域的水质异常，构建起高效率的早期预警雷达网。

# -*- coding: utf-8 -*-
import numpy as np
import matplotlib.pyplot as plt
from scipy.integrate import solve_ivp
from scipy.interpolate import interp1d


def build_time_grid(hours=24, dt_min=5):
    dt_h = dt_min / 60.0
    t = np.arange(0.0, hours + 1e-12, dt_h)
    return t, dt_h


def demand_multiplier(times):
    """双峰日变化，归一化后均值=1。"""
    h = times % 24.0
    raw = (
        0.40
        + 1.20 * np.exp(-0.5 * ((h - 7.5) / 1.8) ** 2)
        + 0.95 * np.exp(-0.5 * ((h - 19.5) / 2.2) ** 2)
    )
    return raw / raw.mean()


def source_chlorine_by_two_component_model(times):
    """
    用二组分模型(3.7~3.9)构建出厂余氯边界条件：
    将清水池简化为CSTR（停留时间 tau_clear）。
    """
    t_knots = np.array([0, 4, 7, 10, 16, 20, 24], dtype=float)
    c_dose_knots = np.array([0.95, 1.02, 1.05, 0.96, 0.92, 0.98, 0.95], dtype=float)
    dose_func = interp1d(t_knots, c_dose_knots, kind="cubic", fill_value="extrapolate")

    tau_clear = 1.0  # h
    k1 = 1.10  # L/(mg*h), 快速组分
    k2 = 0.08  # L/(mg*h), 慢速组分
    c_fast_in = 0.30  # mg/L
    c_slow_in = 0.80  # mg/L

    def rhs(t, y):
        c, c_fast, c_slow = y
        c_inj = float(np.clip(dose_func(t % 24.0), 0.75, 1.20))
        dc = (c_inj - c) / tau_clear - k1 * c * c_fast - k2 * c * c_slow
        dcf = (c_fast_in - c_fast) / tau_clear - k1 * c * c_fast
        dcs = (c_slow_in - c_slow) / tau_clear - k2 * c * c_slow
        return [dc, dcf, dcs]

    y0 = [0.75, c_fast_in, c_slow_in]
    sol = solve_ivp(
        rhs,
        (times[0], times[-1]),
        y0,
        t_eval=times,
        max_step=(times[1] - times[0]),
        rtol=1e-6,
        atol=1e-8,
    )
    return np.clip(sol.y[0], 0.0, None)


def mass_transfer_coeff(v_mph, d_pipe):
    """
    计算 k_f (m/h)，式(3.5)：
    k_f = Sh * D_m / D
    """
    nu = 1.0e-6      # m2/s
    d_m = 1.2e-9     # m2/s (20°C 量级)
    v_mps = v_mph / 3600.0

    re = max(v_mps * d_pipe / nu, 1e-8)
    sc = nu / d_m
    if re < 2300:
        sh = 3.65
    else:
        sh = 0.0149 * (re ** 0.88) * (sc ** 0.33)

    kf = sh * (d_m * 3600.0) / d_pipe  # m/h
    return kf


def simulate_network(
    times,
    c_source,
    q_m3ph,
    temp_c,
    lengths_m,
    diameters_m,
    kw_profile_mph,
    kb20=0.60,
    theta=1.05,
):
    """
    核心：一维对流-反应近似（稳态速度+时变边界）
    C_out(t) = C_in(t-τ) * exp(-k_total * τ)
    k_total = k_b(T) + 2*k_w*k_f/(R*(k_w+k_f))
    """
    n_pipes = len(lengths_m)
    n_nodes = n_pipes + 1
    nt = len(times)

    c = np.zeros((n_nodes, nt), dtype=float)
    c[0, :] = c_source

    kb_t = kb20 * (theta ** (temp_c - 20.0))  # 式(3.3)

    # 初值：按初始工况给一个衰减分布
    q0 = q_m3ph[0]
    for i in range(1, n_nodes):
        d = diameters_m[i - 1]
        area = np.pi * d**2 / 4.0
        v = q0 / max(area, 1e-12)
        tau = lengths_m[i - 1] / max(v, 1e-6)
        c[i, 0] = c[0, 0] * np.exp(-kb_t[0] * tau)

    for j in range(1, nt):
        t = times[j]
        q = q_m3ph[j]

        for i in range(1, n_nodes):
            d = diameters_m[i - 1]
            l = lengths_m[i - 1]
            kw = kw_profile_mph[i - 1]

            area = np.pi * d**2 / 4.0
            v = q / max(area, 1e-12)  # m/h
            tau = l / max(v, 1e-6)    # h

            kf = mass_transfer_coeff(v, d)
            r = d / 2.0
            k_wall = (2.0 * kw * kf) / (r * (kw + kf + 1e-12))  # 式(3.4)
            k_total = kb_t[j] + k_wall  # 式(3.6)

            t_up = t - tau
            c_in = np.interp(
                t_up,
                times[: j + 1],
                c[i - 1, : j + 1],
                left=c[i - 1, 0],
                right=c[i - 1, j],
            )
            c[i, j] = c_in * np.exp(-k_total * tau)

    return c


def calc_kpi(c_mat, times, lengths, diameters, q_m3ph, threshold=0.05):
    terminal = c_mat[-1, :]
    network = c_mat[1:, :]

    # 末梢平均水龄（由瞬时流量估算）
    area = np.pi * diameters**2 / 4.0
    age_terminal = np.sum((lengths[:, None] * area[:, None]) / q_m3ph[None, :], axis=0)

    h = times % 24.0
    night_mask = (h >= 2.0) & (h <= 5.0)

    return {
        "net_mean": network.mean(),
        "terminal_mean": terminal.mean(),
        "terminal_min": terminal.min(),
        "terminal_compliance_pct": 100.0 * np.mean(terminal >= threshold),
        "network_low_pct": 100.0 * np.mean(network < threshold),
        "terminal_age_mean": age_terminal.mean(),
        "terminal_night_min": terminal[night_mask].min(),
    }


def print_kpi_table(kpi_dict):

def main():
    # 1) 时间、负荷、温度
    times, _ = build_time_grid(hours=24, dt_min=5)
    mult = demand_multiplier(times)
    q_m3ph = 72.0 * mult  # 基准20 L/s = 72 m3/h

    # 3) 24段串联系统（25节点）
    lengths = np.array(
        [350, 420, 380, 450, 500, 620, 700, 680, 760, 820, 900, 870,

        dtype=float
    )
    diameters = np.array([0.40] * 8 + [0.30] * 8 + [0.20] * 8, dtype=float)

    kw_mixed = np.linspace(0.08, 0.22, len(lengths))  # m/h
    scenarios = {
        "工况A-全网新管": 0.25 * kw_mixed,

    }

    results = {}
    kpis = {}
    for name, kw_prof in scenarios.items():

            times=times,
            c_source=c_source,
            q_m3ph=q_m3ph,
            temp_c=temp_c,
            lengths_m=lengths,
            diameters_m=diameters,
            kw_profile_mph=kw_prof,
            kb20=0.60,
            theta=1.05,

## 本章小结

1. 管网中余氯的时空演变本质上是反应-对流过程：水体在管段内以活塞流（平推流）形式输移，同时发生与时间呈一阶负指数关系的衰减，即 $C(t) = C_0 e^{-k t}$。
2. 余氯衰减速率系数 $k$ 由体相衰减系数 $k_b$ 和管壁衰减系数 $k_w$ 共同决定，两者分别反映水体本身的化学需氧（有机物、$\mathrm{Fe}^{2+}$等）和管壁生物膜及腐蚀产物的耗氯作用，需通过瓶试验和管段现场试验分别率定。
3. 管段间的水质混合发生在节点处，依据流量守恒假设各上游管段输入的余氯质量通量与总流量之比计算节点混合余氯浓度，构成管网水质模型的边界连接条件。
4. EPANET水质模块采用时间步进的拉格朗日追踪法（水团法），将管段内水体离散为若干"水团"并追踪其在管网中的移动与化学反应，能够高效模拟复杂管网的动态水质变化过程。
5. 余氯不足（低于0.05 mg/L）将导致管网细菌复活风险，而余氯过高（超过0.8 mg/L）则产生消毒副产物（DBPs），两者均威胁饮用水安全，因此水质模型是优化二次加氯策略的核心工具。
