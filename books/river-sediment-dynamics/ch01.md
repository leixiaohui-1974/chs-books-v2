# 第1章 泥沙运动力学基础

## 1.1 本章导读

河流泥沙动力学与河床演变是水利工程、河流地貌学以及水环境科学的交叉学科基础。自然界中的河流在水流的切应力作用下，不断对河床进行冲刷、搬运与沉积，塑造了多姿多彩的地貌形态，同时也对人类的水利开发、防洪减灾及航道整治产生了深远影响。本章作为全书的开篇，系统论述泥沙运动力学的基础理论，涵盖泥沙颗粒的几何与物理特性、泥沙起动机制、悬移质扩散理论以及推移质运动规律。

通过对多相流体力学基本原理的引入，本章试图建立水流与泥沙相互作用的力学图景。读者将从中了解到，泥沙在水中的运动并非简单的颗粒位移，而是受湍流相干结构、重力沉降、颗粒间碰撞等多种因素交织的复杂动力学过程。掌握这些基本规律，是后续章节探讨河道演变模型、水库淤积预测及河网形态动力学的前提条件。本章的理论框架构建将遵循从微观单颗粒受力分析，过渡到宏观群体输移统计规律的逻辑主线，结合必要的数学推导与工程实测数据，为读者提供扎实的力学推演基础。

## 1.2 基本概念与理论框架

### 1.2.1 泥沙颗粒的物理与几何特性

泥沙颗粒的几何尺寸和物理性质直接决定了其在水流中的运动状态。在自然河流中，泥沙颗粒大小相差悬殊，通常采用泥沙粒径分布曲线（级配曲线）来描述泥沙群体的组成。常用的特征粒径包括中值粒径 $d_{50}$、有效粒径 $d_{10}$ 以及表征不均匀程度的分选系数 $S_0 = \sqrt{d_{75}/d_{25}}$。

泥沙的物理性质主要包括颗粒密度 $\rho_s$（石英质泥沙常取 $2650 \text{ kg/m}^3$）、水的密度 $\rho$、干容重 $\gamma_d$ 以及孔隙率 $p$。这些参数在推算河床变形量时起着决定性作用。

单个泥沙颗粒在静水中自由沉降时，受到的重力与流体阻力达到平衡时的终端速度称为沉速（$\omega$）。沉速是表征泥沙在湍流中悬浮能力的特征指标。对于球形细颗粒泥沙，流态处于斯托克斯（Stokes）区（即颗粒雷诺数 $Re_d < 1$），其沉速公式推导基于流体黏性力与有效重力的平衡：

$$ \omega = \frac{(\rho_s - \rho)g d^2}{18 \mu} $$

式中，$g$ 为重力加速度；$\mu$ 为流体动力黏度；$d$ 为泥沙粒径。对于天然非规则形状的粗颗粒泥沙，需考虑形状系数和流体绕流产生的压差阻力，多采用鲁比（Rubey）或张克（Zanke）提出的半经验公式进行计算。

### 1.2.2 泥沙起动与 Shields 曲线

泥沙起动是指床面颗粒在水流剪切力作用下由静止转为运动的临界状态。对床面颗粒进行力学分析，颗粒主要承受重力 $W$、浮力 $F_b$、水流拖曳力 $F_D$ 及上举力 $F_L$。当绕颗粒接触点产生的水流动水力矩大于有效重力的稳定力矩时，颗粒发生滚动或滑动起动。

1936年，Shields 通过量纲分析提出了著名的 Shields 无量纲临界切应力 $\Theta_c$，其表达式为：

$$ \Theta_c = \frac{\tau_c}{(\gamma_s - \gamma)d} $$

式中，$\tau_c$ 为临界起动切应力；$\gamma_s$ 和 $\gamma$ 分别为泥沙和水的容重。Shields 根据大量试验数据，绘制了 $\Theta_c$ 与颗粒雷诺数 $Re_* = \frac{u_{*c} d}{\nu}$ 之间的关系曲线（Shields 曲线）。在层流底层完全掩盖颗粒的水力光滑区（$Re_* < 2$），临界切应力随雷诺数增大而减小；在阻力平方区（水力粗糙区，$Re_* > 500$），$\Theta_c$ 趋于常数，通常取值在 $0.045 \sim 0.06$ 之间。该理论奠定了现代推移质输沙率计算的力学基石。

### 1.2.3 泥沙扩散方程与 Rouse 公式

悬移质泥沙在水流中的垂向分布是紊流掺混与颗粒重力沉降相互抗衡的结果。基于雷诺时均方程及物质质量守恒定律，二维恒定均匀流中的悬移质泥沙扩散方程可表述为：

$$ \frac{\partial C}{\partial t} + u \frac{\partial C}{\partial x} + v \frac{\partial C}{\partial y} + (w - \omega) \frac{\partial C}{\partial z} = \frac{\partial}{\partial x} \left( \epsilon_x \frac{\partial C}{\partial x} \right) + \frac{\partial}{\partial y} \left( \epsilon_y \frac{\partial C}{\partial y} \right) + \frac{\partial}{\partial z} \left( \epsilon_z \frac{\partial C}{\partial z} \right) $$

式中，$C$ 为泥沙体积含沙量；$u, v, w$ 分别为纵向、横向和垂向时均流速；$\epsilon_x, \epsilon_y, \epsilon_z$ 为对应方向的泥沙扩散系数。

针对二维宽浅明渠恒定均匀流，上述方程可简化为垂向扩散项与沉降项的平衡：

$$ \omega C + \epsilon_z \frac{d C}{d z} = 0 $$

根据普朗特混合长理论，假设泥沙扩散系数与水流动量交换系数相同（$\epsilon_z = \nu_t$），且切应力在垂向上呈线性分布 $\tau = \tau_0 (1 - \frac{z}{h})$，可推导出对数流速分布下的动量交换系数 $\nu_t = \kappa u_* z (1 - \frac{z}{h})$。将此代入简化方程并积分，即可得到著名的 Rouse 悬移质垂向分布公式：

$$ \frac{C}{C_a} = \left( \frac{h-z}{z} \frac{a}{h-a} \right)^Z $$

式中，$C_a$ 为参考高度 $a$ 处的含沙量；$h$ 为水深；$Z$ 为悬浮指数（Rouse 数），定义为 $Z = \frac{\omega}{\kappa u_*}$，其中 $\kappa$ 为卡门常数（约 0.4），$u_*$ 为摩阻流速。Rouse 数直观地反映了重力沉降与紊流扩散的相对强弱：$Z$ 越小，泥沙在垂向上分布越均匀；$Z$ 越大，泥沙越集中于近底层。

## 1.3 数学建模与求解方法

河床演变学的核心在于将流体力学控制方程与泥沙连续方程耦合，构建能够描述河床冲淤过程的数学模型。

### 1.3.1 控制方程体系

泥沙运动及其引起的河床变形，遵循流体质量守恒、动量守恒以及泥沙质量守恒原则。以一维明渠流为例，圣维南方程组构成了水流计算的骨架：

连续方程：
$$ \frac{\partial A}{\partial t} + \frac{\partial Q}{\partial x} = 0 $$

动量方程：
$$ \frac{\partial Q}{\partial t} + \frac{\partial}{\partial x}\left( \frac{\beta Q^2}{A} \right) + gA \frac{\partial Z_w}{\partial x} + gA S_f = 0 $$

泥沙连续方程（对流扩散形式）：
$$ \frac{\partial (AC)}{\partial t} + \frac{\partial (QC)}{\partial x} = - \alpha \omega W (C - C_*) $$

河床变形方程（Exner 方程）：
$$ \rho_s (1-p) \frac{\partial Z_b}{\partial t} + \frac{\partial g_b}{\partial x} = \alpha \omega (C - C_*) $$

上述体系中，$A$ 为过水断面面积；$Q$ 为流量；$Z_w$ 为水位；$S_f$ 为摩阻斜率；$C$ 为断面平均含沙量；$C_*$ 为挟沙能力；$\alpha$ 为恢复饱和系数；$W$ 为河宽；$Z_b$ 为河床高程；$g_b$ 为推移质单宽输沙率；$p$ 为床面孔隙率。该方程组揭示了流速场分布决定了泥沙挟沙能力，挟沙能力与实际含沙量的差异驱动了河床的冲刷或淤积，而河床边界的改变反过来又调整了水流场，形成一个动态非线性反馈系统。

### 1.3.2 数值离散与求解算法

由于上述偏微分方程组具有强非线性且在冲淤剧烈时存在激波解（如沙波推移或溯源冲刷），解析解仅限于极其简化的边界条件。工程界普遍采用有限差分法（FDM）或有限体积法（FVM）进行离散求解。

为捕捉激波并抑制数值振荡，对流项通常采用高分辨率的迎风格式，如 TVD（Total Variation Diminishing）格式或 MUSCL-Hancock 格式。由于水流波速与河床演变波速通常存在几个数量级的差异（水流为快变量，河床变形为慢变量），求解策略主要分为解耦求解与全耦合求解两类。工程实践中，为提高计算效率，多采用异步长解耦计算：即在较大的水流时间步长内，保持床面不动求解水动力场；随后更新泥沙浓度场，最后利用较小的变形时间步长更新床面高程。优化理论在网格自适应生成和参数反演校正中同样发挥着作用。

## 1.5 工程启示与应用建议

理论与仿真分析结果为河道治理与水利工程运行提供了科学依据。针对泥沙运动力学的工程实践，提出以下应用建议：

第一，在防洪减灾与航道疏浚工程中，必须顺应河势与泥沙输移规律。由于河流系统的非线性反馈机制，单纯的局部挖泥往往难以长效维持深水航道。工程规划应基于临界起动切应力与 Rouse 垂向分布规律，通过丁坝、顺坝等导治结构调整局部水流场，束水攻沙，利用水流自身动能维持航道水深。

第二，水库调度应引入“蓄清排浑”理念。前文仿真表明，洪峰退水期是落淤的高发时段。针对多泥沙河流上的枢纽工程，在汛期泥沙浓度极高时应采用敞泄运行，借助高流速产生的强大挟沙能力将泥沙排出库区；在非汛期水清时再进行蓄水。这就要求在运行初期利用一维或二维数学模型，准确预测异重流潜入点及挟沙能力的空间分布，优化排沙底孔的开启时机与高程。

第三，在涉水结构物（如桥墩、海上风电基础）的设计中，需充分评估局部冲刷深度。建构筑物的存在会改变局部流线，产生下潜流和马蹄形旋涡。这会导致局部床面剪应力远大于明渠平均剪应力，突破 Shields 临界起动条件。设计者必须依赖精确的三维两相流模型，结合防护抛石的隐蔽工程，确保基础的长期稳定性。

本章探讨的河床冲淤系统不仅是流体力学与地貌学的交叉范畴，在更宏大的系统科学视角下，该方法在水网控制中同样适用。水利系统与本章所述系统在数学结构上存在同构性。将河道视为状态空间，泥沙浓度与床面高程作为状态变量，上游来水来沙及水库调度规则作为控制输入，整个河床演变过程即可抽象为一个具有大迟延、强非线性分布参数的控制系统。

    R = A / P
    Q_est = (1.0 / n) * A * (R ** (2.0 / 3.0)) * np.sqrt(S)
    return Q_est - Q


def solve_flow_depth(Q, B, S, n):
    """给定流量Q，反算矩形断面水深h"""
    return brentq(manning_discharge_residual, 0.05, 20.0, args=(Q, B, S, n))


def settling_velocity_soulsby(d, rho_s, rho_w, nu):
    """Soulsby 经验式：沉速 ws"""
    s = rho_s / rho_w
    d_star = d * (((s - 1.0) * g) / (nu ** 2.0)) ** (1.0 / 3.0)
    ws = (nu / d) * (np.sqrt(10.36 ** 2.0 + 1.049 * d_star ** 3.0) - 10.36)
    return ws


def bedload_mpm(theta, theta_c, d, rho_s, rho_w):
    """Meyer-Peter & Muller 公式：单位宽推移质输沙率 qb (m^2/s)"""
    s = rho_s / rho_w
    excess = np.maximum(theta - theta_c, 0.0)
    phi_b = 8.0 * excess ** 1.5
    qb = phi_b * np.sqrt((s - 1.0) * g * d ** 3.0)
    return qb


def print_kpi_table(rows):
    """打印KPI结果表格"""
    print("\n" + "=" * 72)
    print("KPI结果表（第1章 1.1 泥沙运动力学基础）")
    print("-" * 72)
    print(f"{'指标':<26}{'数值':>24}{'单位':>12}")
    print("-" * 72)
    for name, value, unit, fmt in rows:
        print(f"{name:<26}{fmt.format(value):>24}{unit:>12}")
    print("=" * 72)


# -----------------------------
# 2) 主程序
# -----------------------------
def main():
    # 时间轴与流量过程
    t = np.arange(0.0, sim_hours * 3600.0 + dt, dt)
    Q = (
        Q_mean
        + Q_amp * np.sin(2.0 * np.pi * t / T_flood)
        + Q_sub_amp * np.sin(4.0 * np.pi * t / T_flood + phase_sub)
    )
    Q = np.clip(Q, Q_min, None)

    # 预分配数组
    h = np.zeros_like(t)          # 水深
    u = np.zeros_like(t)          # 平均流速
    tau_b = np.zeros_like(t)      # 床面切应力
    theta = np.zeros_like(t)      # Shields参数
    q_b = np.zeros_like(t)        # 单位宽推移质输沙率

    # 时间步计算：水力 -> 起动 -> 输沙
    for i, Qi in enumerate(Q):
        hi = solve_flow_depth(Qi, B, S0, n_manning)
        A = B * hi
        P = B + 2.0 * hi
        R = A / P

        ui = Qi / A
        tau_i = rho_w * g * R * S0
        theta_i = tau_i / ((rho_s - rho_w) * g * d50)
        qb_i = bedload_mpm(theta_i, theta_c, d50, rho_s, rho_w)

        h[i] = hi
        u[i] = ui
        tau_b[i] = tau_i
        theta[i] = theta_i
        q_b[i] = qb_i

    # Rouse剖面（取峰值流量时刻）
    idx_peak = int(np.argmax(Q))
    H_peak = h[idx_peak]
    u_star_peak = np.sqrt(tau_b[idx_peak] / rho_w)
    ws = settling_velocity_soulsby(d50, rho_s, rho_w, nu)
    P_rouse = ws / (kappa * u_star_peak + 1e-12)

    a_ref = max(0.02, reference_ratio * H_peak)
    z = np.linspace(a_ref, H_peak - 1e-3, 250)
    Cz = C_a * ((((H_peak - z) / z) * (a_ref / (H_peak - a_ref))) ** P_rouse)
    C_mean = simpson(Cz, z) / (H_peak - a_ref)

    # KPI汇总
    q_section = q_b * B
    q_section_cum = cumulative_trapezoid(q_section, t, initial=0.0)
    total_bedload_volume = q_section_cum[-1]
    total_bedload_mass = total_bedload_volume * rho_s
    transport_ratio = np.mean(theta >= theta_c) * 100.0
    tau_c = theta_c * (rho_s - rho_w) * g * d50

    kpi_rows = [
        ("平均流量", np.mean(Q), "m^3/s", "{:.2f}"),
        ("平均水深", np.mean(h), "m", "{:.3f}"),
        ("平均流速", np.mean(u), "m/s", "{:.3f}"),
        ("峰值床面切应力", np.max(tau_b), "Pa", "{:.3f}"),
        ("临界切应力", tau_c, "Pa", "{:.3f}"),
        ("峰值Shields参数", np.max(theta), "-", "{:.4f}"),
        ("起动时长占比", transport_ratio, "%", "{:.1f}"),
        ("峰值单位宽输沙率", np.max(q_b), "m^2/s", "{:.6e}"),
        ("总推移质体积", total_bedload_volume, "m^3", "{:.2f}"),
        ("总推移质质量", total_bedload_mass, "kg", "{:.2e}"),
        ("峰值Rouse数", P_rouse, "-", "{:.3f}"),
        ("峰值时刻平均浓度", C_mean, "-", "{:.6f}"),
    ]
    print_kpi_table(kpi_rows)

    # -----------------------------
    # 3) 绘图
    # -----------------------------
    t_h = t / 3600.0
    fig, axs = plt.subplots(2, 2, figsize=(12, 8))

    # 图1：流量-水深过程
    ax = axs[0, 0]
    ax.plot(t_h, Q, color="tab:blue", lw=1.8, label="流量 Q")
    ax.set_xlabel("时间 (h)")
    ax.set_ylabel("流量 (m^3/s)", color="tab:blue")
    ax.tick_params(axis="y", labelcolor="tab:blue")
    ax.grid(alpha=0.3)
    ax2 = ax.twinx()
    ax2.plot(t_h, h, color="tab:orange", lw=1.8, label="水深 h")
    ax2.set_ylabel("水深 (m)", color="tab:orange")
    ax2.tick_params(axis="y", labelcolor="tab:orange")
    ax.set_title("流量与水深过程")

    # 图2：切应力与Shields参数
    ax = axs[0, 1]
    ax.plot(t_h, tau_b, color="tab:red", lw=1.8, label="床面切应力 τb")
    ax.axhline(tau_c, color="k", ls="--", lw=1.2, label="临界切应力 τc")
    ax.set_xlabel("时间 (h)")
    ax.set_ylabel("切应力 (Pa)")
    ax.grid(alpha=0.3)
    ax2 = ax.twinx()
    ax2.plot(t_h, theta, color="tab:green", lw=1.5, label="Shields θ")
    ax2.axhline(theta_c, color="tab:purple", ls="--", lw=1.2, label="临界 θc")
    ax2.set_ylabel("Shields参数 (-)")
    ax.set_title("起动判别过程")

    # 图3：推移质输沙率与累计输沙量
    ax = axs[1, 0]
    ax.plot(t_h, q_b, color="tab:brown", lw=1.8, label="单位宽输沙率 qb")
    ax.set_xlabel("时间 (h)")
    ax.set_ylabel("qb (m^2/s)")
    ax.grid(alpha=0.3)
    ax2 = ax.twinx()
    ax2.plot(t_h, q_section_cum, color="tab:cyan", lw=1.8, label="累计输沙体积")
    ax2.set_ylabel("累计输沙体积 (m^3)")
    ax.set_title("推移质输运过程")

    # 图4：Rouse浓度剖面
    ax = axs[1, 1]
    ax.plot(Cz / C_a, z / H_peak, color="tab:olive", lw=2.0)
    ax.set_xlabel("相对浓度 C(z)/Ca")
    ax.set_ylabel("相对水深 z/H")
    ax.grid(alpha=0.3)
    ax.set_title(f"峰值流量时Rouse剖面 (P={P_rouse:.2f})")

    fig.suptitle("第1章1.1 泥沙运动力学基础仿真", fontsize=13)
    plt.tight_layout()
    plt.show()


if __name__ == "__main__":
    main()

代码解读（约800字）  
这份脚本把“泥沙运动力学基础”拆成四个教学层次：水动力计算、起动判别、推移质输运和悬移质分布。先看水动力部分，代码用矩形断面和Manning公式建立流量与水深关系。由于Manning公式里水深是隐变量，脚本使用 `scipy.optimize.brentq` 对每个时刻的流量反求水深，这一步体现了“理论框架中的数值求解思想”。拿到水深后，按断面面积与流量求平均流速，并用 `τb=ρgRS` 求床面切应力，这构成泥沙受力分析的基础输入。  
第二层是Shields起动理论。脚本按 `θ=τb/[(ρs-ρ)gd]` 计算无量纲起动参数，再与临界值 `θc` 比较。这样就把“何时开始动床”从定性描述变成了可量化判别。代码同时给出临界切应力 `τc`，并在图中画阈值线，便于课堂讲解“超临界与亚临界”两种工况。KPI中的“起动时长占比”正是把这个判别结果做成统计指标，可直接用于章节作业或方案比选。  
第三层是推移质输沙。脚本采用Meyer-Peter & Muller公式：当 `θ>θc` 时按超临界强度计算单位宽输沙率 `qb`，否则取零。这个处理保持了物理意义上的门槛特征，也与教材中“起动-输运耦合”逻辑一致。随后把 `qb` 乘河宽得到断面输沙率，再用 `cumulative_trapezoid` 积分得到累计输沙体积与总质量。这样不仅能看瞬时峰值，还能看全过程输沙总量，符合工程分析“峰值+总量并重”的需求。  
第四层是Rouse悬移质框架。脚本先用Soulsby公式估算沉速 `ws`，再按 `P=ws/(κu*)` 求Rouse数，其中 `u*` 来自峰值时刻切应力。最后根据Rouse剖面公式计算 `C(z)`，并用 `simpson` 求平均浓度。若 `P` 较大，浓度沿深度衰减更快，表示颗粒更难悬浮；`P` 较小则说明紊动支撑更强。通过这一段，学生可以把“沉速-紊动-垂向分布”三者关系直观看清。  
图形上，脚本给了四联图：流量与水深、切应力与Shields、输沙率与累计量、Rouse剖面。KPI表则把平均水深、峰值切应力、起动占比、峰值输沙率、总输沙量等核心结果集中输出，便于报告引用。整体上这是一个“可运行、可解释、可扩展”的最小教学模型：若后续进入1.2和1.3，可在此基础上继续加入河床演变方程、粒径分组和参数敏感性分析。
