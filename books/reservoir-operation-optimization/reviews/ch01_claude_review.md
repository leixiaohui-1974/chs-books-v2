## 二、详细评审意见

### 2.1 P1级问题详解

#### 问题1: 逆时序推算公式不严谨 (第70行)

**位置**: 1.2.1节,第70行

**问题描述**:
```
V_k = V_{dead} - [ ... ] Δt
```

这个公式有两个问题:
1. 通式里把下一时段状态固定成`V_dead`,但实际应该是`V_{k+1}`
2. 尾水位函数`Z_{tail}(Q)`和水头损失`Δh`在推导中被简化掉了,但前文(第64行)明确定义了它们

**修改建议**:
```
V_k = V_{k+1} + [ ̄I_k - Q_k - ̄E_k ] Δt

其中Q_k需满足:
N_{gar} = A_c · η · Q_k · H_k
H_k = (Z_k + Z_{k+1})/2 - Z_{tail}(Q_k) - Δh
```

并明确说明这是一个关于`V_k`的隐式方程,需要迭代求解。

#### 问题2: 保证出力线包络方向错误 (第71行)

**位置**: 1.2.1节,第71行

**问题描述**:
```
"多组枯水序列推算出的包络线的下外包线"
```

这个表述是错误的。为了满足保证出力约束,应该取**上包络**(逐时段最大值),而不是下包络。

**物理意义**:
- 不同枯水序列推算出不同的最低水位线
- 为了在**所有**枯水情况下都能保证出力,必须取最高的那条线(上包络)
- 取下包络会降低安全裕度,无法保证在最不利情况下的出力

**修改建议**:
```
"多组枯水序列推算出的包络线的**上外包线**(逐时段最大值),即构成最终的保证出力线。"
```

#### 问题3: 动态汛限表述有合规风险 (第161行)

**位置**: 1.4.2节,第161行

**问题描述**:
```
"允许水位阶段性浮动至防洪限制水位之上"
```

这个表述容易误导工程实践。动态汛限水位必须经过严格的风险评估和水行政主管部门批复,不能随意突破。

**修改建议**:
```
"在经过严格的风险评估和水行政主管部门批复的前提下,可实施动态汛限水位控制,在预报未来无大洪水时,允许水位在批复的动态汛限规则范围内浮动,但必须满足防洪安全约束。"
```

### 2.2 P2级问题详解

#### 问题4: 水量平衡方程不一致

**位置**: 第34行 vs 第81行

**问题描述**:
- 第34行包含渗漏项`S(t)`: `dV/dt = I - Q - E - S`
- 第81行省略了渗漏项: `V_{t+1} = V_t + (I_t - Q_t - E_t) Δt`

**修改建议**:
在第81行前增加假设说明:
```
"为简化模型,以下优化模型中忽略渗漏项S_t(通常渗漏量远小于入库流量和下泄流量,对优化结果影响有限)。"
```

#### 问题5: 出力公式单位体系未闭合

**位置**: 第63行

**问题描述**:
```
A_c为重力加速度等综合常数(通常取9.81)
```

这个表述不严谨:
1. `A_c ≈ 9.81`的量纲是什么?
2. 如何从`m³/s × m`得到`MW`?

**修改建议**:
```
"A_c为综合系数,当Q单位为m³/s,H单位为m,N单位为MW时,A_c = ρg/10⁶ ≈ 9.81×10⁻³ MW·s/(m⁴),其中ρ为水密度(1000 kg/m³),g为重力加速度(9.81 m/s²)。"
```

#### 问题6: DP缺少终端条件说明

**位置**: 第98行

**问题描述**:
动态规划递推方程缺少:
1. 终端条件`f_{T+1}(V_{T+1})`的定义
2. 状态离散化方法
3. "全局最优"应限定为"离散网格上的全局最优"

**修改建议**:
补充:
```
"终端条件: f_{T+1}(V_{T+1}) = 0

状态离散: 将库容区间[V_min, V_max]离散为N个网格点,网格间距Δ V = (V_max - V_min)/N。

注意: DP求得的是离散状态空间上的全局最优解,连续空间的真实最优解需通过插值逼近。"
```
