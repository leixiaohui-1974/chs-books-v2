## 二、ch02评审总结 (Codex主评)

### 2.1 识别问题 (5个)

#### 问题1: SDP递推式状态定义不闭合 ⚠️ 高优先级

**位置**: ch02.md:85, 92

**问题描述**:
- SDP方程中期望项使用了 p_{jk}^t,但未显式绑定"当前处于 I_t=i^j"
- 决策域写成 Ω_R(S_t),未体现对当前来水状态 I_t 的依赖
- 严格写法应是 Ω_R(S_t,I_t) 或 Ω_R(S_t,i^j)

**修复建议**:
```markdown
决策可行域应写为:
$$\Omega_R(S_t, I_t) = \{R_t \mid S_{min} \leq S_{t+1} \leq S_{max}, R_{min} \leq R_t \leq R_{max}\}$$

SDP递推方程应明确当前来水状态:
$$F_t(S_t, i^j) = \max_{R_t \in \Omega_R(S_t, i^j)} \left\{ B_t(S_t, R_t) + \alpha \sum_{k=1}^{K} p_{jk}^t F_{t+1}(S_{t+1}, i^k) \right\}$$
```

---

#### 问题2: 折现型Bellman与平均收益漂移判据混用 ⚠️ 高优先级

**位置**: ch02.md:95, 97, 99

**问题描述**:
- 先给出 α∈(0,1] 的折现型SDP
- 再使用含 ΔF 的稳态漂移收敛判据
- 后者通常对应平均收益/相对价值框架(α=1特例)
- 两套理论未分场景说明,严格性不足

**修复建议**:
```markdown
### 2.3.2 收敛判据的两种框架

**折现型框架** (α < 1):
- 收敛判据: ||F^{(n+1)} - F^{(n)}||_∞ < ε
- 适用场景: 有限时域优化,重视近期收益

**平均收益框架** (α = 1):
- 收敛判据: |ΔF^{(n+1)} - ΔF^{(n)}| < ε (相对价值迭代)
- 适用场景: 无限时域稳态优化,长期平均效益

本章采用折现型框架(α=0.95),确保迭代收敛性。
```

#### 问题3: 逆向水量平衡仅部分完成 ⚠️ 中优先级

**位置**: ch02.md:38, 45

**问题描述**:
- 前向平衡式量纲与符号方向正确(已修复)
- 但未显式写出逆向形式 R_t = f(S_t, S_{t+1}, I_t, ...)
- 未给出隐式项 E_t(S_t, S_{t+1}) 的数值求解闭环
- 缺少唯一性/迭代停止准则如何嵌入DP的说明

**修复建议**:
```markdown
### 2.2.1 状态转移方程的正向与逆向形式

**正向形式** (给定S_t和R_t,求S_{t+1}):
$$S_{t+1} = S_t + (I_t - R_t - E_t) \Delta t$$

**逆向形式** (给定S_t和S_{t+1},求R_t):
$$R_t = \frac{S_t - S_{t+1}}{\Delta t} + I_t - E_t(S_t, S_{t+1})$$

其中蒸发项 E_t(S_t, S_{t+1}) 依赖于平均水面面积,需迭代求解:
1. 初始猜测: R_t^{(0)} = I_t
2. 计算: S_{t+1}^{(k)} = S_t + (I_t - R_t^{(k)} - E_t^{(k)}) \Delta t
3. 更新: R_t^{(k+1)} = (S_t - S_{t+1,target}) / \Delta t + I_t - E_t^{(k)}
4. 收敛判据: |R_t^{(k+1)} - R_t^{(k)}| < 0.01 m³/s

在DP中,每个状态转移都需执行上述迭代。
```

#### 问题4: R_t/Q_spill/R_max语义边界歧义 ⚠️ 中优先级

**位置**: ch02.md:38, 44, 52, 58

**问题描述**:
- 状态方程写成 R_t + Q_spill,t 同时扣减
- 收益项又写 min(R_t, R_max) 且"超出R_max作为弃水"
- 容易导致"R_t是总下泄还是仅可控下泄"的双重解释

**修复建议**:
```markdown
### 符号说明

| 符号 | 含义 | 单位 | 约束 |
|------|------|------|------|
| R_t | 可控下泄流量(过机流量) | m³/s | R_min ≤ R_t ≤ R_max |
| Q_spill,t | 溢流量(不可控) | m³/s | Q_spill,t ≥ 0 |
| Q_total,t | 总下泄流量 | m³/s | Q_total,t = R_t + Q_spill,t |
| R_max | 机组最大过流能力 | m³/s | 设计参数 |

**状态转移方程**:
$$S_{t+1} = S_t + (I_t - R_t - Q_{spill,t} - E_t) \Delta t$$

**溢流判定**:
$$Q_{spill,t} = \max(0, \frac{S_t + (I_t - R_t - E_t)\Delta t - S_{max}}{\Delta t})$$

**发电效益** (仅计可控下泄):
$$B_t = A_c \cdot \eta \cdot R_t \cdot H_t \cdot \Delta t$$
```

#### 问题5: 缺少伪代码,实现与公式不一致 ⚠️ 中优先级

**位置**: ch02.md:179, figures/ch02_simulation.py:179, ch02.md:92

**问题描述**:
- 章节有长代码块,但缺少教材友好的算法伪代码
- SDP公式含 α,示例实现却是 `val = reward + exp_future` (未乘α)

**修复建议**:
```markdown
### 算法2.1: 随机动态规划(SDP)求解单库调度

**输入**:
- 状态空间: S = {S_1, S_2, ..., S_N}
- 来水状态: I = {i^1, i^2, ..., i^K}
- 转移概率: P = {p_{jk}^t}
- 折现因子: α ∈ (0,1]
- 收敛阈值: ε

**输出**: 最优策略 π*(S_t, I_t) 和价值函数 F*(S_t, I_t)

**算法步骤**:
```
1. 初始化: F^(0)(S_t, i^j) = 0, ∀S_t, i^j
2. 迭代 n = 1, 2, ...:
   For each S_t ∈ S:
     For each i^j ∈ I:
       For each R_t ∈ Ω_R(S_t, i^j):
         计算即时收益: B_t(S_t, R_t)
         计算状态转移: S_{t+1} = f(S_t, R_t, I_t)
         计算期望未来价值: V = α · Σ_k p_{jk}^t · F^(n-1)(S_{t+1}, i^k)
         更新: Q(S_t, i^j, R_t) = B_t + V
       F^(n)(S_t, i^j) = max_R Q(S_t, i^j, R_t)
       π^(n)(S_t, i^j) = argmax_R Q(S_t, i^j, R_t)
3. 收敛检查: If ||F^(n) - F^(n-1)||_∞ < ε, 停止
4. 返回: π*, F*
```

**Python实现修正** (figures/ch02_simulation.py:179):
```python
# 错误: val = reward + exp_future
# 正确:
val = reward + alpha * exp_future  # 必须乘折现因子
```
```
