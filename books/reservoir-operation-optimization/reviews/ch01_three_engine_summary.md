### 2.1 P1级问题 (Critical - 必须修复)

#### P1-1: 逆时序计算公式错误 (line 70)

**问题描述**:
```markdown
V_{t-1} = V_t - (Q_{in,t} - Q_{out,t} - E_t) \cdot \Delta t
```

**错误原因**: 逆时序递推时,水量变化的符号应该反向。从t时刻推t-1时刻,应该是"减去流入,加上流出"。

**正确公式**:
```markdown
V_{t-1} = V_t + (Q_{out,t} - Q_{in,t} + E_t) \cdot \Delta t
```

**修复位置**: ch01.md:70

---

#### P1-2: 包络线方向描述错误 (line 71)

**问题描述**:
> "从终点向起点逐步收缩可行域"

**错误原因**: 逆时序DP的包络线是从终点向起点**扩展**可行域,而非收缩。每个时段的可行域是下一时段所有可行状态的前驱集合的并集。

**正确表述**:
> "从终点向起点逐步构建可行域包络线,每个时段的可行域由下一时段所有可行状态的可达前驱集合决定"

**修复位置**: ch01.md:71

```python
# 附录A: 水库DP调度Python实现
# 文件: reservoir_dp.py

import numpy as np

def reservoir_dp(Q_in, V_init, V_target, Z_V_curve, Q_V_curve,
                 V_min, V_max, Q_min, Q_max, T, dt=1.0):
    """
    水库动态规划调度

    参数:
        Q_in: 入库流量序列 [m³/s], shape=(T,)
        V_init: 初始库容 [10⁸m³]
        V_target: 目标库容 [10⁸m³]
        Z_V_curve: 水位-库容关系函数 Z=f(V)
        Q_V_curve: 出力-库容关系函数 P=g(V,Q)
        V_min, V_max: 库容约束 [10⁸m³]
        Q_min, Q_max: 出流约束 [m³/s]
        T: 调度时段数
        dt: 时段长度 [h]

    返回:
        V_opt: 最优库容过程 [10⁸m³], shape=(T+1,)
        Q_opt: 最优出流过程 [m³/s], shape=(T,)
        P_total: 总发电量 [MWh]
    """
    # 离散化状态空间
    V_grid = np.linspace(V_min, V_max, 50)
    N_V = len(V_grid)

    # 初始化DP表
    J = np.full((T+1, N_V), np.inf)  # 代价函数
    policy = np.zeros((T, N_V), dtype=int)  # 策略

    # 终端条件
    J[T, :] = (V_grid - V_target)**2 * 1e6  # 惩罚偏离目标

    # 逆时序递推
    for t in range(T-1, -1, -1):
        for i, V_t in enumerate(V_grid):
            # 可行出流范围
            Q_out_min = max(Q_min, (V_t - V_max + Q_in[t]*dt*3.6e-3) / (dt*3.6e-3))
            Q_out_max = min(Q_max, (V_t - V_min + Q_in[t]*dt*3.6e-3) / (dt*3.6e-3))

            if Q_out_min > Q_out_max:
                continue  # 不可行状态

            # 离散化决策空间
            Q_out_grid = np.linspace(Q_out_min, Q_out_max, 20)

            for Q_out in Q_out_grid:
                # 状态转移
                V_next = V_t - (Q_in[t] - Q_out) * dt * 3.6e-3  # 10⁸m³

                # 插值计算J[t+1, V_next]
                J_next = np.interp(V_next, V_grid, J[t+1, :])

                # 即时效益 (负发电量,因为求最小)
                P_t = Q_V_curve(V_t, Q_out)  # MW
                stage_cost = -P_t * dt  # MWh (负值)

                # Bellman方程
                J_candidate = stage_cost + J_next

                if J_candidate < J[t, i]:
                    J[t, i] = J_candidate
                    policy[t, i] = Q_out  # 存储最优决策

    # 前向模拟最优轨迹
    V_opt = np.zeros(T+1)
    Q_opt = np.zeros(T)
    V_opt[0] = V_init

    for t in range(T):
        # 插值查找最优决策
        i = np.argmin(np.abs(V_grid - V_opt[t]))
        Q_opt[t] = policy[t, i]
        V_opt[t+1] = V_opt[t] - (Q_in[t] - Q_opt[t]) * dt * 3.6e-3

    P_total = -J[0, np.argmin(np.abs(V_grid - V_init))]

    return V_opt, Q_opt, P_total

# 使用示例

    V_init = 221.5  # 初始库容 221.5亿m³
    V_target = 221.5
    V_min = 165.0  # 死库容
    V_max = 393.0  # 正常蓄水位库容
    Q_min = 6000  # 最小下泄流量
    Q_max = 100000  # 最大泄流能力

    # 简化的水位-库容关系 (实际应用需用实测曲线)
```
