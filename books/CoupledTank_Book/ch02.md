# 第2章 双容水箱的物理与数学模型

## 2.1 什么是双容水箱系统？（水网微缩模型）

双容水箱系统由两个有蓄水容积的节点及其连接通道构成，是研究液位控制最经典的“最小复杂系统”。
它比单水箱更接近真实水网，因为它同时包含：

1. 储能环节：两个液位状态 $h_1, h_2$。
2. 传输环节：箱间流量 $q_{12}$。
3. 执行环节：阀门/泵对入流或出流的调节。
4. 扰动环节：需求变化、泄漏、背压波动。

因此，双容系统既足够简单，便于推导和教学；又足够复杂，能够体现耦合、非线性与约束，是“从理论到代码”最合适的桥梁对象。
这也是映射矩阵把 Ch2 绑定到 `core/coupled_tank/dynamics.py` 和 ODE 求解链路的原因。

## 2.2 质量守恒定律：一维流体动力学的基础

双容系统动力学首先来自质量守恒。若液体密度近似常量、截面积固定，则体积守恒可直接写成液位微分方程：

$$A_1\frac{dh_1}{dt}=q_{in}-q_{12}-q_{l1}$$
$$A_2\frac{dh_2}{dt}=q_{12}-q_{out}-q_{l2}$$

其中：

1. $A_1, A_2$：两箱等效截面积。
2. $q_{in}$：入口流量（受泵/阀控制）。
3. $q_{12}$：箱间耦合流量。
4. $q_{out}$：出口流量。
5. $q_{l1}, q_{l2}$：泄漏或未建模损失项。

若箱间流量由液位差驱动，可采用孔口/阻力近似：
$$q_{12}=C_{12}a_{12}\sqrt{2g\max(h_1-h_2,0)}$$
出口可写为：
$$q_{out}=C_{o}a_{o}\sqrt{2gh_2}$$

以上就是 `core/coupled_tank/dynamics.py` 应显式实现的核心：
“以守恒为骨架，以非线性流量关系为肌理，形成状态导数计算函数”。
对应测试 `tests/coupled_tank/test_ch02_mass_balance.py` 的目标，就是验证该守恒关系在离散仿真中仍成立（允许数值误差范围内）。

## 2.3 阀门、泵与管道的非线性特征

真实系统不是线性叠加系统，主要非线性来自执行器与流量通道。

常见非线性来源如下：

1. 阀门开度-流量非线性：低开度区灵敏、高开度区趋缓。
2. 泵特性曲线：扬程与流量相关，控制增益随工况变化。
3. 压差驱动平方根关系：流量对液位差呈 $\sqrt{\Delta h}$ 变化。
4. 饱和与死区：阀门最小可控开度、最大流量上限。
5. 方向切换与单向约束：某些通道仅允许单向流动。

因此在 `core/coupled_tank/dynamics.py` 中，工程上通常需要具备：

1. 输入限幅（saturation）以保证物理可实现性。
2. 非负液位约束，避免数值积分导致“负水位”。
3. 条件分支处理单向流或反向阻断。
4. 参数化接口，支持标定与场景切换。

这部分决定了模型是否“像真实系统”。
如果只保留线性近似，控制器离线可用，但上线后常出现模型失配、调节漂移与异常振荡。

## 2.4 状态空间方程：用数学语言描述水箱（ODE求解）

将第2.2节与第2.3节整合，可得到非线性状态模型：
$$\dot{x}=f(x,u,d,\theta),\quad x=[h_1,h_2]^T$$
其中：

1. $u$：可控输入（泵速、阀门开度）。
2. $d$：扰动输入（需求突变、泄漏、来水波动）。
3. $\theta$：参数集合（面积、流量系数、损失参数）。

在某个稳态点 $(x^*,u^*)$ 附近，可线性化得到：
$$\delta\dot{x}=A\delta x + B\delta u + E\delta d$$
这为后续 PID 整定、MPC 预测模型、可控可观分析提供统一接口。

数值实现时，需要 ODE 求解器把连续系统离散推进。无论使用显式欧拉、改进欧拉还是 RK 系列，核心要求有三点：

1. 数值稳定：步长与对象时间常数匹配，避免假振荡。
2. 物理一致：积分后状态仍满足边界与守恒逻辑。
3. 可复现：同一输入应得到一致轨迹，便于测试追溯。

这正对应 Ch2 的两组关键测试：

1. `tests/coupled_tank/test_ch02_mass_balance.py`：验证守恒与收支闭合。
2. `tests/coupled_tank/test_ch02_ode_stability.py`：验证积分过程稳定、轨迹有界、无非物理发散。

同时，映射矩阵中的 `coupled_tank_server.simulate_dynamics` 将动力学与求解器封装为可调用能力，使“书本方程”可被直接执行为“实验曲线”。

## 本章小结

第2章完成了从物理直觉到数学模型的过渡：

1. 双容水箱是水网控制问题的最小可研究系统。
2. 质量守恒给出状态演化主方程。
3. 阀泵管非线性决定模型的工程真实性。
4. 状态空间与 ODE 求解把理论转为可运行代码。
