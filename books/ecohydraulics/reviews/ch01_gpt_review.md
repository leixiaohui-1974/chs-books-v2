### P1 重要问题

1. **章节编号体系异常（D5 编号连续性）**
   - 本章采用"1. 学习目标 / 2. 教材理论 / 3. 案例分析 / 4. 本章小结"的阿拉伯数字体例，与其他章节（"## 1.1 / 1.2"）的体例存在根本性差异。
   - 建议：全书统一章内节级体例，推荐采用"1.1 / 1.2 / 1.3"层次，与工程类教材标准一致。

2. **图片路径混用（D2 图表完整性）**
   - 第1节包含两个概念图：一个使用 `![...](assets/ch01/...)` 正常语法，一个使用 `![...](assets/ch01/...)` 反斜杠语法，同章内两种语法共存，渲染结果不一致。
   - 建议：统一为无反斜杠格式。

3. **Tennant法数据核验（D1 技术准确性）**
   - 案例中计算MAR = 63.6 m³/s，由12个月数据（12+15+35+60+85+110+150+130+80+45+25+15）/12 = 763/12 = 63.58 m³/s，**计算正确**。
   - 但仿真结果表中"Poor"列（生存底线）全年均为6.4 m³/s（10% MAR = 6.36 ≈ 6.4），与Tennant原始论文（产卵期Poor为10% MAR，非产卵期也为10%）一致，数值正确。
   - 问题在于：案例中提到公司方案"恰好等于MAR的10%（即6.4 m³/s）"，而公司实际方案为"6.5 m³/s"，6.5/63.6=10.2%，严格而言略高于10%，文中"恰好等于"表述不够准确。
   - 建议：修改为"接近MAR的10%（约10.2%），属于Poor级别"。

4. **参考文献缺失（D4 参考文献）**
   - 本章引用了大量方法（Tennant法、P-III型分布、IHA法），但未提供独立的参考文献列表。必须引用：
     - Tennant, D.L. (1976) 原始文献（*Fisheries*, ISSN 0363-2415）
     - Richter, B.D. et al. (1996) IHA方法原始文献（*Conservation Biology*）
     - 中国P-III型分布水文学规范（GB/T 22482）
   - 建议：在本章末增加参考文献列表，至少8条。

### P2 改进建议

1. **IHA公式符号说明（D3 公式正确性）**
   - 水文变异度公式 $DHA_j = \frac{|O_j - E_j|}{E_j} \times 100\%$ 中，$E_j$ 标注为"期望值（通常取中位数）"，但数学"期望值"与"中位数"含义不同，会引起混淆。
   - 建议：统一使用"参考值"或"建坝前历史中位数"，并在公式中改用 $M_j$ 表示中位数。

2. **P-III型分布公式参数定义（D3）**
   - 式(1.1)中 $\beta$ 为"尺度参数"，但P-III型分布在中国水文手册中习惯以Cv（变差系数）和Cs（偏态系数）参数化，而非 $(\alpha, \beta, a_0)$，两种参数化形式的转换关系未说明。
   - 建议：补充P-III参数与Cv、Cs的换算关系，便于读者与工程实践对接。

3. **CHS框架融合（D6）**
   - 第2.4节将生态基流纳入CHS（水系统控制论）约束框架，分层讨论L0/L2/L3的约束机制，内容新颖，但CHS框架术语（六元受控系统 $\Sigma$）未在本章前面介绍，对读者不友好。
   - 建议：增加脚注说明CHS框架来源，或将此节移至全书框架章之后。

4. **"水力倒挂"描述的措辞（D6）**
   - 案例结果分析中"如果你是一个不懂水力学的环保主义者"措辞偏主观，不适合教材用语。
   - 建议：改为"这展示了单纯水文学方法的局限性：当天然枯水期流量本身低于生态标准时，必须引入水库调度弹性机制"。

5. **表格数字精度（D5）**
   - 仿真结果表中Poor列 = 6.4 m³/s，而 MAR×10% = 6.358 m³/s，保留一位小数四舍五入为6.4，精度一致。但Good列 = 25.4 m³/s（MAR×40% = 25.43），Excellent = 31.8 m³/s（MAR×50% = 31.78），均合理，无误。

### 评审结论

本章在8本书中教学设计最为出色，采用"提出实际工程问题→理论工具→案例计算→工业建议"的案例驱动模式，有效降低了抽象理论的学习门槛。Tennant法、P-III分布、IHA三种方法的对比表格是亮点。主要问题是**参考文献完全缺失**（Tennant原始文献必须引用）和**章内节号体例与全书不统一**。IHA公式中$E_j$定义混用需修正。

**优先修复顺序**：
1. 补充参考文献列表（Tennant 1976, Richter et al. 1996等）
2. 修复图片渲染语法（去除反斜杠）
3. 统一章内节号体例至全书标准
4. 修正IHA公式中"期望值"表述
