# 《综合能源系统仿真与优化》：多能互补的数学引擎

## 大纲规划
**字数目标**：5万字以上
**读者定位**：能源规划工程师、运筹学算法研究员
**核心导向**：冷、热、电、气多能流耦合建模与混合整数线性规划（MILP）。

### 第一部分：系统认知与多能流网络
**第1章：什么是综合能源系统（IES）？**
1.1 能源孤岛的打破：电、热、气、冷的物理耦合
1.2 能源集线器（Energy Hub）建模概念
1.3 IES 的典型设备：CCHP、热泵、电制气（P2G）

### 第二部分：物理设备与能流建模
**第2章：多能转换设备建模**
2.1 燃气轮机与微型燃气轮机的热电联产（CHP）模型
2.2 吸收式制冷机与电制冷机的 COP 曲线
2.3 电锅炉与热泵的非线性效率转换

**第3章：储能与管网时空动态建模**
3.1 储电、储热、储气设备状态空间模型
3.2 供热管网的热阻力与热延时（水力-热力耦合）
3.3 天然气管网的管存效应与流量方程

### 第三部分：优化调度算法与 MILP
**第4章：微电网日前经济调度优化**
4.1 目标函数构建：运行成本、碳排放与弃风弃光惩罚
4.2 约束条件线性化：设备爬坡、启停约束与 Big-M 法
4.3 混合整数线性规划（MILP）求解器（Gurobi/CPLEX）接入

**第5章：多主体博弈与分布式优化**
5.1 IES 参与电力市场的交易机制
5.2 纳什均衡（Nash Equilibrium）在能源博弈中的应用
5.3 交替方向乘子法（ADMM）实现分布式隐私求解

### 第四部分：仿真平台与案例
**第6章：基于 Python 的 IES 仿真平台搭建**
6.1 面向对象的 IES 组件类设计（Class Architecture）
6.2 拓扑图生成与数据流转图
6.3 零碳工业园区的四季能流优化实战分析
