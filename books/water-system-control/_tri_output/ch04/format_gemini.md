# 第 4 章：串级与前馈：抵御复杂水网扰动的护城河

## 1. 学习目标
本章探讨当单回路 PID 无法应对水系统内部极其严重的非线性和外部剧烈扰动时，如何通过改变控制器拓扑结构来实现降维打击。
读者需要掌握：
1. 为什么“单回路 PID”在控制复杂的非线性阀门时经常失效。
2. 串级控制（Cascade Control）的主副双环架构及其物理意义。
3. 流量内环（Secondary Loop）对泵站或管网压力波动的强效抑制作用。
4. 前馈控制（Feedforward Control）的超前感知思想与实现。

## 2. 教材理论：为 PID 引入辅助控制回路
在第 2 章中，我们应用了单回路 PID 来控制水箱液位：测量水位偏差，计算出控制指令并直接作用于执行机构（水阀）。此即为**单回路控制（Single-Loop Control）**。
然而，在真实的水务现场，阀门并非理想的线性执行机构。
1. **死区与非线性**：许多大型蝶阀在 $0 \sim 10\%$ 的开度区间内存在死区（无流量输出）；在 $10 \sim 30\%$ 区间内流量急剧增加；而在 $80 \sim 100\%$ 区间内流量趋于饱和（流量特性曲线呈现严重非线性）。
2. **外部扰动（Disturbance）**：假设市政管网突发失压（如管网负荷剧增），即使阀门开度维持在 $50\%$ 不变，流入水箱的实际流量也会瞬间大幅下降。

若仅依靠单一 PID 控制液位，由于水箱截面积较大，水位变化表现出极大的惯性延迟。当主控制器检测到液位偏差并进而增加阀门开度时，系统往往已经产生了显著的滞后。

为解决上述难题，控制工程中引入了**串级控制（Cascade Control）**。
其核心思想为：**主控制器负责宏观系统目标的追踪，而将局部扰动抑制和非线性补偿交由副控制器处理。**
- **主控制器（液位 PID）**：其控制目标为维持系统设定液位。其输出不再是直接的阀门开度指令，而是一个**目标流量设定值**（例如：设定所需进水流量为 $1.5 m^3/s$）。
- **副控制器（流量 PID）**：其控制目标为精确跟踪主控制器下发的流量设定值。其设定值（Setpoint）即为主控制器的输出（$1.5 m^3/s$）。副控制器直接调节执行机构（阀门开度），以克服阀门非线性与管网压力波动等局部扰动，确保实际流量稳定于 $1.5 m^3/s$。

**物理优势**：流量环（内环）具有极快的动态响应特性。当市政管网突发失压导致进水流量减小时，内环流量 PID 能够在数秒内检测到偏差并迅速开大阀门进行补偿。在这一扰动引起水位显著下降之前，内环控制系统已将其迅速且有效地抑制。

## 3. 案例分析：理论与实践的桥梁（管网失压扰动下的串级控制仿真）

### 3.1 案例背景 (Context)
某智慧水厂的清水池液位由前端的加压泵和调节阀共同控制。在每日早晨的用水高峰期（$t=200 \sim 300s$），前端市政管网压力会发生剧烈波动与跌落。
若采用传统的单回路 PID（依据液位误差直接控制阀门开度），由于清水池截面积巨大，每次失压均会导致严重的液位跌落与超调。工程要求将系统升级为“液位-流量”串级控制架构，并在不改变物理设备的前提下，通过算法仿真验证其对管网失压扰动的抑制能力。

### 3.2 问题描述 (Problem)
- **水箱参数**：底面积 $A = 2.0 m^2$，目标恒定水位 $SP = 5.0m$。
- **阀门非线性**：存在 $10\%$ 的死区，且流量特性呈现严重非线性。
- **失压扰动**：在 $200s \le t \le 300s$ 期间，管网供水能力突降至额定状态的 $60\%$。
- **对比实验**：
  1. 系统 A：单回路 PID 控制（液位误差 $\to$ 阀门开度）。
  2. 系统 B：串级 PID 控制（液位误差 $\to$ 流量指令 $\to$ 流量误差 $\to$ 阀门开度）。

### 3.3 解题思路 (Solution Approach)
本研究构建了一个包含严重非线性阀门和管网动态跌落的仿真模型：
1. **非线性阀门建模**：编写具备死区和指数响应特性的自定义阀门数学模型。
2. **单回路部署**：部署响应缓慢的液位 PID（由于积分作用显著），直接驱动非线性阀门。
3. **串级部署**：
   - 部署主回路 PID，输出带限幅的流量设定点（$SP_{flow}$）。
   - 部署高增益、无积分或弱积分的副回路 PID，基于瞬时流量误差 $e = SP_{flow} - Q_{actual}$ 高频调节阀门开度。
4. **扰动注入与解算**：在 $t=200s$ 时引入阶跃扰动，使得实际流量 $Q_{actual}$ 衰减至正常值的 $60\%$，对比观察两种控制架构的抗扰动性能。

### 3.5 实验验证与结果剖析 (Verification & Result Interpretation)
通过仿真对比，串级控制系统的结构优势得到了充分验证：
- **单回路系统应对扰动的滞后性**：在 $t=210s$（管网失压后 $10s$），由于流量锐减（由表中数据可知跌至 $0.806 m^3/s$），单回路系统未能及时响应，导致水箱液位显著下降至 $3.547m$。单回路 PID 需要积累高达 $1.5m$ 的液位偏差后，控制器才输出显著的调节作用以增加阀门开度。
- **串级系统内环的快速抗扰特性**：在管网失压的瞬间，串级系统的内环（流量 PID）表现出极高的灵敏度，迅速且大幅度地调整阀门开度。这使得进水流量在 $t=210s$ 时迅速恢复并稳定在 $1.107 m^3/s$。**内环在极短时间内有效抑制了供水压力波动的扰动，使得外环（液位）未受到显著影响。** 此时串级控制下的液位维持在 $4.164m$。
- **非线性免疫**：阀门的死区和极度非线性特性被有效地包含并补偿于内环控制之中。对于外环主控制器而言，广义的被控对象展现出具有理想线性响应特征的“按需供水”系统。

### 3.6 工业部署与运行建议 (Industrial Deployment Recommendations)
1. **配置高精度流量计**：应避免依赖阀门开度来间接估算流量作为串级内环的反馈信号。在复杂的市政管网中，固定的阀门开度可能因管网负荷变化而对应差异巨大的实际流量。必须在水箱入口配置高精度的电磁流量计或超声波流量计，这是构建有效串级控制系统的硬件基础。
2. **内外环的整定原则**：串级系统的参数整定必须遵循“先内后外”的原则。首先将主回路置于手动模式，专注于内环（流量环）的参数整定，使其具备快速响应能力（允许适度振荡，通常采用较大的比例增益 $P$ 和较小的积分增益 $I$）。内环调试完成后，它将有效屏蔽内部非线性与局部扰动；随后切换至自动模式进行外环（液位环）整定。外环参数整定应侧重于稳定性和无静差跟踪（通常设定较大的积分时间）。若内外环响应频带过于接近，系统极易引发强烈的耦合振荡。
