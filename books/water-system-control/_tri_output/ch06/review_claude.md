# 第 6 章：多变量最优控制：LQR 算法与权重博弈

## 1. 学习目标
本章探讨当系统中存在多个相互牵制的变量（如水位、水压、塔身震动）时，如何利用线性二次型调节器（LQR）实现全局最优的控制策略。
读者需要掌握：
1. 状态空间方程（MIMO 系统）与 PID 控制（SISO 系统）的维度差异。
2. 线性二次型调节器（LQR）的物理意义与代价函数（Cost Function）。
3. 代数黎卡提方程（CARE）的求解逻辑。
4. Q 矩阵（状态惩罚）与 R 矩阵（控制惩罚）的工程权衡（Trade-off）。

## 2. 教材理论：要水位还是要命？
传统的 PID 控制器是“单输入单输出（SISO）”的：它只盯着一个误差（比如水位），然后输出一个动作（阀门）。
但在现代巨型水利工程中，系统往往是“多输入多输出（MIMO）”的。
举个极端的例子：**高耸的倒立摆式水塔**。
水塔建在极细的柱子上。当你猛开阀门向上抽水时，巨大的水流冲力（反作用力）会导致高塔发生摇晃（甚至倒塌）。此时系统有两个相互矛盾的状态：
1. **水位状态（Level）**：供水局希望你尽快把水打满（需要猛开阀门）。
2. **结构状态（Tilt Angle）**：建筑局希望水塔绝对不能摇晃（需要极慢地开阀门）。

面对这种矛盾，PID 彻底无能为力。因为只要它一发力追求水位，就会引发灾难性的结构震动。

我们需要一种具有“全局大局观”的控制算法——**线性二次型调节器（LQR）**。
LQR 不会死盯某一个变量，而是把系统所有的状态（$x_1, x_2, \dots$）和所有的输入（$u_1, u_2, \dots$）打包成一个矩阵。然后它定义了一个**代价函数（Cost Function, $J$）**：
$$ J = \int_0^\infty (x^T Q x + u^T R u) dt $$
- $Q$ 是状态权重矩阵。你把哪个状态的权重设得高，LQR 就会拼命把这个状态压回 0。
- $R$ 是输入权重矩阵。如果你把 $R$ 设得高，LQR 就会极其吝啬地使用阀门（省电且平滑）。

LQR 算法的魔力在于，只要你给定了物理系统的矩阵 $A, B$ 和你想要的权重矩阵 $Q, R$，它就能通过求解复杂的**代数黎卡提方程（CARE）**，在几毫秒内直接吐出一组极其完美、且在数学上绝对稳定的全状态反馈增益矩阵 $K$。
控制定律极其简单：$u = -Kx$。
