# 第 9 章 滑模变结构控制（SMC）

## 1. 学习目标
本章探讨当系统中存在连模型预测控制（MPC）和卡尔曼滤波（KF）都难以准确估算的极端参数漂移与未知扰动时，如何利用一种极其有效且具备强鲁棒性的非线性算法来控制系统。
读者需要掌握：
1. 模型不确定性（Model Uncertainty）与外部扰动（Disturbance）对线性控制器的严重影响。
2. 滑模控制（Sliding Mode Control, SMC）的滑模面设计与趋近律。
3. 抖振（Chattering）现象的物理根源及其工程消除方法（边界层法）。
4. SMC 在水处理加药、污水生化反应等复杂工况下的应用价值。

## 2. 理论基础：滑模变结构控制原理
前几章探讨的 MPC 和 LQR 虽然性能优异，但存在一个显著弱点：**它们对模型精度要求极高**，高度依赖数学模型的准确性。
当管道内部结垢导致摩擦阻力突增 $50\%$，或水箱发生不可测泄漏时，LQR 会计算出错误的增益，MPC 会产生偏差极大的预测。在这些情况下，它们不仅无法实现有效控制，甚至可能导致系统失稳。

面对严重模型失配与参数漂移的复杂工况，控制学界引入了**滑模变结构控制（SMC）**。
SMC 的控制核心在于其极强的鲁棒性：无论模型误差或外部扰动多大，只要系统满足一定边界条件，控制器就能强制状态轨迹趋向并保持在预设的滑模面上。

它的核心设计分为两步：
1. **定义滑模面（Sliding Surface, $s$）**：
   在状态空间中划定一个超平面，例如 $s = c \cdot e + \dot{e}$。当系统状态处于该面上时，误差 $e$ 会自动呈指数衰减至 $0$。只要系统维持在滑模面上，即可实现稳定控制。
2. **切换控制（Switching Control）**：
   这是 SMC 区别于其他控制方法的核心特征。它通常通过符号函数 $u = -K \cdot \text{sgn}(s)$ 进行控制。当状态偏离滑模面时（如 $s > 0$ 或 $s < 0$），控制器通过快速的极性反转（如全关或全开执行器），强制系统状态回归并保持在滑模面上。这种高频切换机制使得只要控制增益 $K$ 足够大，SMC 即可克服外部扰动，实现极强的“鲁棒性（Robustness）”。

**抖振的代价**：
在实际物理系统中，执行器存在机械惯性和响应频率限制。这种高频切换控制会导致执行器产生剧烈振荡（即抖振，Chattering），进而可能严重缩短机电设备的使用寿命。在工程应用中，通常通过引入一个**边界层（Boundary Layer, $\Phi$）**，将非连续的符号函数柔化为连续的饱和函数（Saturation），以微小的稳态误差为代价，有效减轻执行器的机械磨损。

## 3. 案例分析：理论与实践的桥梁（模型漂移与强干扰下的液位控制）

### 3.1 案例背景
以某工况复杂的工业废水混合池为例。其底部的出水阀门受到强酸腐蚀，导致流量系数 $C$ 极不稳定，随时间呈现正弦波动（在 $0.25$ 到 $0.75$ 之间变化，导致标称模型严重失配）。
此外，在 $t \in [25, 35]\text{s}$ 期间，上游系统会突然汇入大量未经计量的废水，形成剧烈的未测外部扰动。
在此工况下，采用常规 PID 控制器的混合池极易发生失控乃至溢流事故。为此，本案例将设计一种基于边界层法的滑模控制器以实现系统的稳定控制。

### 3.2 问题描述
- **标称模型**（控制器内部参考模型）：水箱底面积 $A=2.0$，流量系数 $C=0.5$。
- **真实系统**（实际物理过程）：水箱底面积 $A=2.0$，流量系数 $C(t) = 0.5(1 + 0.5\sin(0.1t))$（呈现显著时变漂移）。
- **外部扰动**：在 $t \in [25, 35]\text{s}$ 期间，施加 $d(t) = 2.0 \text{ m}^3/\text{s}$ 的不可测额外注入流量。
- **SMC 设计**：采用等效控制 $u_{eq}$ 消除标称动态，辅以鲁棒切换控制 $u_{sw} = K_{sw} \cdot \text{sat}(s/\Phi)$ 抑制参数漂移和扰动。设定切换增益 $K_{sw} = 3.0$，边界层厚度 $\Phi = 0.1$。
目标为对比最优整定的 PID 与 SMC 在该复杂工况下的追踪表现。

### 3.3 解题思路
本案例构建了模拟系统真实非线性演进与控制算法的仿真环境：
1. **系统真实状态**：物理更新方程选用 $\dot{x} = \frac{u - C_{\text{true}} \sqrt{x} + d_{\text{dist}}}{A}$ 引入时变漂移参数。
2. **等效控制（标称模型补偿）**：SMC 算法内部计算 $u_{eq} = C_{\text{nom}} \sqrt{x}$，用于抵消标称动态。
3. **切换控制（鲁棒补偿）**：SMC 计算当前状态至滑模面的距离 $s = SP - x$。若 $|s| > \Phi$，输出饱和控制量；若 $|s| \le \Phi$，则在边界层内进行线性平滑过渡。
4. **对比分析**：在完全相同的参数漂移与扰动条件下，同时运行 PID 与 SMC 进行性能对比评价。

### 3.6 工业部署建议
1. **优势应用场景**：在生物污水处理（如曝气池溶解氧 DO 控制）、加氯消毒等**参数易因环境或生化反应而发生剧烈改变**的工况中，SMC 具备显著优势。SMC 降低了对高精度机理模型的依赖，仅需确定系统不确定性与扰动的上界，即可通过设计合理的鲁棒增益实现对时变参数与未知扰动的有效抑制。
2. **执行器保护与工程实现**：在实际工程应用中需注意，SMC 的高鲁棒性在一定程度上依赖于控制指令的高频切换。在实际部署至 PLC 之前，必须在控制器输出端增加一阶低通滤波器，或合理放宽边界层厚度 $\Phi$。应坚决避免将含有剧烈阶跃的理想 SMC 控制信号直接作用于变频器或执行阀，以防止设备因高频振荡引发过载故障甚至停机。
