# 《水资源规划与管理》：算水账的宏观博弈

## 大纲规划
**字数目标**：5万字以上
**读者定位**：水利规划师、流域管理者、环境经济学家
**核心导向**：水资源供需平衡分析、水权分配模型与生态水文学宏观规划。

### 第一部分：算水账的基础
**第1章：全球变暖下的水资源挑战**
1.1 水资源承载力与“三条红线”控制
1.2 气候变化对径流年际/年内分配的扰动
1.3 农业灌溉、工业用水与生态基流的冲突

**第2章：供需平衡分析（Water Balance）**
2.1 长系列水文资料插补与频率分析（P-III 型曲线）
2.2 需水预测模型：定额法、宏观经济模型与系统动力学（SD）
2.3 二次供水与中水回用的资源核算

### 第二部分：水库群规划与水权分配
**第3章：梯级水库群的中长期优化调度**
3.1 水库的兴利调节与防洪库容动态控制（汛期限制水位）
3.2 动态规划（DP）与随机动态规划（SDP）在水库调度中的应用
3.3 多目标进化算法（NSGA-II）求解发电与供水的帕累托前沿

**第4章：水权交易与博弈分配**
4.1 初始水权分配原则：现状优先 vs 公平效率
4.2 水资源价值的经济学评估（影子价格）
4.3 跨流域调水工程（如南水北调）的生态补偿博弈

### 第三部分：生态与环境约束
**第5章：生态需水保障与水质规划**
5.1 Tennant 法与逐月最小生态基流计算
5.2 纳污能力计算与水功能区限制排污总量
5.3 面源污染控制与海绵城市规划效益评估

### 第四部分：现代数字规划工具
**第6章：数字孪生流域与智能规划**
6.1 WEAP（水评估与规划系统）模型原理与 Python 接入
6.2 基于大模型（LLM）的水资源政策文本自动解析与合规检查
6.3 极端干旱情景下的系统韧性（Resilience）压力测试
