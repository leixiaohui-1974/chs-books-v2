# 第4章 能-粮耦合建模

## 本章导读

随着全球人口激增与经济快速发展，能源与粮食系统的交互关系日益复杂。现代农业的高产出高度依赖于化石能源与电力的持续投入，同时生物质能的开发又使得能源生产与粮食生产在土地、水资源等要素上产生直接竞争。这种双向互馈机制构成了能-粮耦合系统的物理基础。本章作为《水-能-粮纽带系统建模》的第4章，系统探讨能-粮耦合的理论框架与数学建模方法。内容涵盖农业生产全生命周期的能源流溯源、生物质能开发对粮食安全的冲击机制、以及能源价格波动向农产品市场的传导路径。通过引入多目标优化理论与系统动力学方法，本章旨在构建能-粮耦合系统的定量描述模型，并通过工程仿真分析揭示系统在不同情景下的演化规律，为跨部门协同规划与水土资源精细化管理提供理论支撑与计算工具。

## 4.1 基本概念与理论框架

本节从系统工程的视角，解析农业系统能量代谢的基本规律，探讨能源与农业在物理空间与经济维度上的耦合互馈机制。

### 4.1.1 农业生产的能源投入特征
现代农业系统本质上是一个通过耗散外部能量来维持低熵状态的开放系统。能源投入贯穿于农业生产的全过程，可分为直接能源投入与间接能源投入两类。直接能源投入指农机具运行消耗的燃油、农田灌溉与温室环境控制消耗的电力等；间接能源投入则隐含在化肥、农药、农膜等农业生产资料的制造与运输环节中。

根据能量守恒定律与物质代谢理论，农业系统的能量转化效率（Energy Return on Investment, EROI）决定了系统的长效稳定性。伴随农业集约化程度的提升，单位播种面积的能源密度显著增加。以华北平原典型的冬小麦-夏玉米轮作系统为例，其化肥投入（特别是高耗能的氮肥）所折算的间接能耗通常占总能耗的50%以上，而地下水超采区深层机井灌溉的电力消耗亦占据较高比重。表4-1给出了典型农作物生产全生命周期的能耗结构特征，数据为区域统计均值。

### 4.1.2 生物质能与粮食的资源竞争机制
生物质能作为可再生能源的重要分支，在替代化石燃料、降低碳排放方面具有明确的战略价值。然而，第一代生物燃料（如以玉米为原料的燃料乙醇、以大豆和油菜籽为原料的生物柴油）的大规模开发，直接触发了“与人争粮、与粮争地”的矛盾。

该竞争机制在空间尺度上表现为耕地利用方式的强互斥性。在区域给定的耕地红线约束下，用于种植能源作物（如甜高粱、柳枝稷）的土地面积扩张必然挤压粮食作物的播种面积。在资源约束层面，水利灌溉资源的分配也面临类似博弈。从经济学视角考察，当原油价格超过生物燃料生产的盈亏平衡点时，能源市场的套利驱动将促使大量农产品从传统食物链流向工业产业链，从根本上改变全球农产品的供需平衡边界。第二代（非粮纤维素）和第三代（微藻）生物燃料虽然在理论上缓解了直接竞争，但其商业化进程受制于较高的转化成本和技术壁垒，在模型评估的短期周期内难以完全消除系统的结构性张力。

### 4.1.3 能源价格向农产品市场的传导路径
能源市场与农产品市场的价格联动是能-粮耦合在经济维度的核心呈现形式。该传导路径具有多重性与非线性滞后特征。

首先是成本推动型传导。原油与天然气价格上涨直接推高农机作业成本与化肥（特别是合成氨工艺）制造成本。由于农产品需求的价格弹性通常较低，生产要素成本的增加在经历一定滞后期后，将作为刚性约束转化为农产品终端价格的上涨。

其次是需求拉动型传导。如前所述，高油价刺激生物燃料产能扩张，导致玉米、甘蔗等基础农产品作为能源原材料的需求激增。这种跨界需求的涌入打破了传统农产品市场的封闭均衡态，使得农产品价格具备了与能源资产同构的金融属性。

此外，宏观经济参数（如通货膨胀率、汇率波动）也在两大市场的交互中发挥放大或阻尼作用。建立包含原油价格、化肥指数、生物燃料产量与粮食价格的多变量协同演化模型，是量化该传导机制的基础。

## 4.2 数学建模与求解方法

本节建立能-粮耦合系统的定量描述模型。我们采用系统动力学（System Dynamics, SD）结合多目标优化（Multi-Objective Optimization）框架，将物理过程与经济规律内化于微分方程组与代数约束条件中。

### 4.2.1 农业能源需求模型
农业能源需求函数 $E_{agri}(t)$ 由各个生产环节的能耗累加而成。假设系统内存在 $N$ 种作物，第 $i$ 种作物的播种面积为 $A_i(t)$，其单位面积的直接能耗与间接能耗分别为 $e_{d,i}(t)$ 和 $e_{ind,i}(t)$。总能源需求模型表示为：

$$
E_{agri}(t) = \sum_{i=1}^{N} A_i(t) \cdot \left[ e_{d,i}(t) + e_{ind,i}(t) \right]
$$

在水利设施完备的农业区，灌溉电力消耗 $E_{pump}(t)$ 是直接能耗的主要构成，主要受地下水埋深 $H(t)$、灌溉水量 $W_{irr}(t)$ 及水泵机组综合效率 $\eta$ 影响：

$$
E_{pump}(t) = \frac{\rho_w g H(t) W_{irr}(t)}{3.6 \times 10^6 \cdot \eta}
$$

式中，$\rho_w$ 为水的密度（$1000 \, \text{kg/m}^3$）；$g$ 为重力加速度（$9.81 \, \text{m/s}^2$）；常数 $3.6 \times 10^6$ 用于将焦耳转换为千瓦时；$E_{pump}(t)$ 的单位为 $\text{kWh}$。

间接能耗中，化肥生产的折算能耗 $E_{fert}(t)$ 由施肥总量 $F_k(t)$ 与各类肥料（$k \in \{N, P, K\}$）的工业能量转化系数 $\lambda_k$ 共同决定：

$$
E_{fert}(t) = \sum_{k \in \{N, P, K\}} \lambda_k F_k(t)
$$

### 4.2.2 生物质能产量与耕地竞争模型
定义系统总可用耕地面积为 $A_{total}$，划分为粮食作物用地 $A_f(t)$ 和能源作物用地 $A_e(t)$。严格的空间约束方程为：

$$
A_f(t) + A_e(t) \le A_{total}
$$

生物质能的产出热值 $B(t)$ 取决于能源作物的种植面积、单产水平 $Y_e(t)$ 以及单位质量作物的基准热值系数 $\theta_e$：

$$
B(t) = A_e(t) \cdot Y_e(t) \cdot \theta_e \cdot (1 - \delta_c)
$$

式中，$\delta_c$ 表征生物质在物理化学转化（如发酵制取乙醇）过程中的能量散失率。
为满足区域基本口粮安全，系统必须始终受制于最低粮食需求红线 $D_f(t)$ 的硬约束：

$$
A_f(t) \cdot Y_f(t) \cdot (1 - \gamma_L) \ge D_f(t)
$$

其中，$Y_f(t)$ 为粮食系统单产，$\gamma_L$ 为粮食在储运环节的综合损耗率。

### 4.2.3 价格传导的动态方程
采用误差修正模型（Error Correction Model, ECM）描述原油价格 $P_{oil}(t)$ 对粮食价格 $P_{food}(t)$ 的动态冲击与传导。设两者在长期存在协整关系 $P_{food} = \alpha + \beta P_{oil} + \varepsilon$。反映短期价格动态变化的方程为：

$$
\Delta P_{food}(t) = \phi_0 + \sum_{j=1}^{p} \phi_j \Delta P_{food}(t-j) + \sum_{k=0}^{q} \psi_k \Delta P_{oil}(t-k) + \Omega \cdot ECM_{t-1} + u_t
$$

式中，$\Delta$ 为差分算子；$ECM_{t-1} = P_{food}(t-1) - \alpha - \beta P_{oil}(t-1)$ 为前一期的非均衡状态误差项；$\Omega$ 为误差修正系数，反映系统偏离长期均衡状态后向其回归的内在调整力度，物理意义上要求 $\Omega < 0$；$u_t$ 为白噪声随机扰动序列。

### 4.2.4 多目标优化问题构建
在统筹区域粮食安全与能源收益的宏观决策下，系统的运筹表现为典型的多目标优化问题（MOP）。目标函数涵盖整个周期内系统净经济效益最大化 $J_1$ 与生态净耗能最小化 $J_2$。

$$
\max J_1 = \int_{0}^{T} e^{-r t} \left[ P_{food}(t) Y_f(t) A_f(t) + P_{bio}(t) B(t) - C_{agri}(t) \right] dt
$$

$$
\min J_2 = \int_{0}^{T} \left( E_{agri}(t) - B(t) \right) dt
$$

式中，$r$ 为资金折现率；$P_{bio}(t)$ 为生物质能单位热值的市场出清价格；$C_{agri}(t)$ 涵盖化肥、水资源、机械作业等农业生产总成本。
核心决策变量为逐年演进的耕地分配比例 $A_f(t), A_e(t)$ 以及化肥与灌溉水资源的投入强度。

### 4.2.5 求解算法
对于上述含有微分-代数系统约束的动态多目标优化模型，常采用数值近似方法进行降维。本章应用基于时间离散的配点法（Collocation Method），将连续型动态优化转化为有限维的非线性规划（NLP）问题。离散化后的模型调用改进的带精英策略非支配排序遗传算法（NSGA-II）执行求解。在算法迭代中，针对约束条件违规程度引入自适应惩罚函数，引导种群向可行域快速收敛。通过提取 Pareto 前沿，为决策者提供经济收益与能耗相干性的定量权衡界面。

## 4.4 工程启示与应用建议

建立在严密理论建模与大规模并行仿真分析基础上，针对现代农业水利工程网的宏观规划、项目论证与运行调度提出以下落地建议。

### 4.4.1 构建多能互补的农业微电网集群
针对深层水灌溉能耗持续攀升的硬件痛点，农田水利工程规划应摈弃单一的“拉电打井”模式，转而构建“光-风-储-水”耦合的多能互补微型电网架构。利用干渠沿线及骨干蓄水工程水面布设分布式光伏组件，通过智能控制柜将光伏瞬态出力曲线与作物灌溉日内需水过程进行高保真匹配。在水网数学模型设计中，须将既往集中式大电网的无限容量边界，修正为具备孤岛运行与黑启动能力的微电网非线性潮流约束方程群。

### 4.4.2 划定基于能量代谢强度的水土开发红线
现行土地利用红线多聚焦于空间面积底线与土壤基础理化性状。本章模型揭示，在深层地下水漏斗区及高扬程梯级提水灌区开展集约化农业，从全域生命周期能源成本角度进行评估是处于劣势的。规划管理部门应深度融合GIS空间分析与SD模型，编制区域“农业综合能量代谢等值线图”。对测算 EROI 指数长期低于盈亏临界值的区划，应实施退耕休耕、旱作改造或生态补水，通过空间重构压降系统整体的基础能耗。

### 4.4.3 推广生物质资源内循环与梯级利用工程
为规避第一代生物质能“与粮争地”的结构性矛盾，工程设计的发力点应锚定农业生态系统的物质闭路循环。规模化建设集中式大型沼气发酵工程或生物质气化发电站，以农作物秸秆、根茬、畜禽排泄物作为主导原料替代能源专作种植。在系统建模中，该策略表现为引入废弃物综合回收率参数，对生物质能产出方程 $B(t)$ 进行解耦重构，使其逐步摆脱对土地面积 $A_e(t)$ 的绝对绑定。工程实践层面需配套构建高密度的秸秆收储运管网与沼液精准还田输水系统。

### 4.4.4 完善能源与粮价联动的数字化防灾预警

# -----------------------------

SIM_YEARS = 20                  # 仿真年限
N_POINTS = 241                  # 输出点数量（约每月一个点）
TIME_SPAN = (0.0, float(SIM_YEARS))

# 初始状态
E0 = 420.0                      # 初始农业可用能源（PJ）
F0 = 500.0                      # 初始粮食产量（Mt）

def merge_params(base, override):
    """合并参数字典。"""
    p = dict(base)

    return p


def nexus_ode(t, y, p):
    """
    能-粮耦合模型：
    dE/dt = invest + feedback*F - depr*E
    dF/dt = r*F*(1-F/K) + eff*max(E-demand,0) - penalty*max(demand-E,0) - climate*F
    demand = e(t)*F, e(t)=e0*exp(-tech*t)
    """
    E, F = y
    E_eff = max(E, 0.0)
    F_eff = max(F, 0.0)

    # 单位粮食能耗随技术进步指数下降
    ei_t = p["e0"] * np.exp(-p["tech"] * t)

    # 粮食系统能源需求
    demand = ei_t * F_eff
    net_energy = E_eff - demand

    # 能源子系统动力学
    dE = p["invest"] + p["feedback"] * F_eff - p["depr"] * E_eff

    # 粮食子系统动力学（内生增长 + 能源盈余/短缺影响 - 气候损失）
    growth = p["r"] * F_eff * (1.0 - F_eff / p["K"])
    bonus = p["eff"] * max(net_energy, 0.0)
    loss = p["penalty"] * max(-net_energy, 0.0) + p["climate"] * F_eff
    dF = growth + bonus - loss

    return [dE, dF]


def simulate_scenario(name, params):

    t_eval = np.linspace(TIME_SPAN[0], TIME_SPAN[1], N_POINTS)
    y0 = [params["E0"], params["F0"]]

    sol = solve_ivp(
        fun=lambda t, y: nexus_ode(t, y, params),
        t_span=TIME_SPAN,
        y0=y0,
        t_eval=t_eval,
        method="RK45",
        rtol=1e-6,
        atol=1e-8
    )
    if not sol.success:
        raise RuntimeError(f"{name} 仿真失败：{sol.message}")

    t = sol.t
    E = sol.y[0]
    F = sol.y[1]

    # 衍生量计算
    ei = params["e0"] * np.exp(-params["tech"] * t)
    demand = ei * np.maximum(F, 0.0)
    shortage = np.maximum(demand - E, 0.0)
    gap_rate = shortage / (demand + 1e-9)
    security = E / (demand + 1e-9)  # 能源保障率（>1表示供给充足）

    # 耦合协调度 D（常见构造）
    Ue = (E - E.min()) / (E.max() - E.min() + 1e-9)
    Uf = (F - F.min()) / (F.max() - F.min() + 1e-9)
    C = 2.0 * np.sqrt(Ue * Uf) / (Ue + Uf + 1e-9)
    T = 0.5 * (Ue + Uf)
    D = np.sqrt(C * T)

    kpi = {
        "scenario": name,
        "food_final": float(F[-1]),
        "food_mean": float(np.mean(F)),
        "security_final": float(security[-1]),
        "gap_mean": float(np.mean(gap_rate)),
        "shortage_cum": float(np.trapz(shortage, t)),
        "intensity_mean": float(np.mean(demand / (np.maximum(F, 1e-9)))),
        "coord_final": float(D[-1])
    }

    return {
        "name": name,
        "t": t,
        "E": E,
        "F": F,
        "demand": demand,
        "gap_rate": gap_rate,
        "security": security,
        "D": D,
        "kpi": kpi
    }

这段脚本对应“能-粮耦合建模”的核心思想是：把能源系统与粮食系统视作两个相互作用的状态变量，并通过连续时间动力学方程描述它们的反馈关系。状态量分别是农业可用能源 \(E(t)\) 与粮食产量 \(F(t)\)。模型采用两条微分方程：\(\frac{dE}{dt}=I+\phi F-\delta_E E\)，表示能源供给由基础投入 \(I\) 增长，同时受到系统折损 \(\delta_E E\) 抑制，并考虑粮食副产物回收形成的正反馈 \(\phi F\)；\(\frac{dF}{dt}=rF(1-\frac{F}{K})+\eta\max(E-eF,0)-\xi\max(eF-E,0)-\delta_FF\)，其中 \(rF(1-F/K)\) 是粮食系统内生增长，\(K\) 为资源约束下的上限。式中最关键的是能源约束项：粮食系统能源需求为 \(e(t)F\)，而单位粮食能耗设为 \(e(t)=e_0e^{-\lambda t}\)，反映技术进步引起的能耗下降。若能源盈余 \(E-eF>0\)，则通过 \(\eta\) 转化为增产；若能源短缺 \(eF-E>0\)，则通过惩罚系数 \(\xi\) 形成减产。  
实现层面上，脚本使用 `scipy.integrate.solve_ivp` 求解常微分方程，时间跨度为20年，241个采样点近似月度分辨率。`simulate_scenario` 负责单情景计算，除得到 \(E,F\) 轨迹，还派生出能源需求、缺口量、缺口率与保障率。这里的缺口率定义为 \(\max(eF-E,0)/(eF)\)，能直接衡量能源瓶颈强度。为了体现教材中的综合评价思想，脚本引入耦合协调度 \(D\)：先将 \(E,F\) 归一化得到 \(U_E,U_F\)，再计算耦合度 \(C=\frac{2\sqrt{U_EU_F}}{U_E+U_F}\)，综合指数 \(T=0.5(U_E+U_F)\)，最终 \(D=\sqrt{CT}\)。该指标越接近1，说明能粮系统越协同。  
KPI表格部分没有依赖额外库，而是手工对齐输出，保证在最小依赖条件下可运行。指标包括末期与平均粮食产量、末期能源保障率、平均能源缺口率、累计能源缺口、平均单位粮食能耗和末期耦合协调度，既覆盖“产量-效率”，也覆盖“安全-协调”。图形部分给出四个视角：多情景粮食演化、基准情景能源供需对比、耦合协调度轨迹、能源缺口率轨迹。这样可快速识别三类机制：技术进步是否真正降低单位能耗、可再生强化是否提升保障率、以及系统是否从“高波动耦合”走向“稳定协同”。  
三种情景设计也具有教学意义：基准情景用于标定系统常态；效率提升情景通过提高 \(\lambda\) 和降低短缺惩罚，模拟节能工艺与管理优化；可再生强化情景通过提高投资与回馈系数，模拟生物质利用和分布式能源部署。对比KPI后，读者可以直接讨论“高投入路径”与“高效率路径”的差异，进而衔接第4章中关于参数识别、政策模拟和鲁棒性分析的扩展内容。
