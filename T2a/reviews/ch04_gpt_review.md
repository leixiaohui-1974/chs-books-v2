## P1 重要问题（强烈建议修复）

1. **Muskingum传递函数H(z)分母符号需核实**
   - 式(4-13)：H(z) = (C₀z+C₁)/(z-(-C₂)) 即分母写为 z+C₂
   - 原差分方程(4-10)：Q_out,k+1 = C₀Q_in,k+1 + C₁Q_in,k + C₂Q_out,k
   - 对差分方程取z变换：z·Q_out(z) = C₀·z·Q_in(z) + C₁·Q_in(z) + C₂·Q_out(z)
   - 整理：Q_out(z)·(z - C₂) = Q_in(z)·(C₀z + C₁)
   - 因此：H(z) = (C₀z + C₁)/(z - C₂)
   - **文中写法**：z-(-C₂) = z+C₂，即分母为(z+C₂)
   - **正确形式**：分母应为(z-C₂)，不是(z+C₂)
   - 需要确认C₂的符号：C₂ = [K(1-X) - 0.5Δt] / [K(1-X) + 0.5Δt]，在稳定条件2KX≤Δt≤2K(1-X)下，若Δt < 2K(1-X)则C₂>0
   - **若C₂>0**，文中z+C₂ = z-(-C₂)，将C₂写为负值来凑成z-(-C₂)的形式，但传递函数分母应为(z-C₂)，极点在z=C₂（正实轴），系统稳定（|C₂|<1时）
   - **实际问题**：文中式(4-13)的第二种写法"(C₀+C₁z⁻¹)/(1+C₂z⁻¹)"换算为z域是(C₀z+C₁)/(z+C₂)，这意味着极点在z=-C₂（负实轴），与z变换结果(z-C₂)矛盾
   - **结论**：式(4-13)的z⁻¹形式应为(C₀+C₁z⁻¹)/(1-C₂z⁻¹)，不是(1+C₂z⁻¹)
   - **修复方案**：将式(4-13)从"(C₀+C₁z⁻¹)/(1+C₂z⁻¹)"改为"(C₀+C₁z⁻¹)/(1-C₂z⁻¹)"

2. **五级模型层级在ch01与ch04中标注不完全一致**
   - ch01式(1-1)：LSV→IDZ→ID→I→SS（5级，箭头说明为"空间集总/去零点/去延迟/稳态"）
   - ch04式(4-5)：LSV→IDZ→ID→I→SS（同5级，但每个箭头的标注不同："空间集总化/略去零点/略去延迟/稳态"）
   - 表述基本一致，仅措辞微小差异（"去零点"vs"略去零点"），属于P2级别
   - 但LSV的全称在ch01是"线性化Saint-Venant"，ch04也应明确说明，两处措辞统一即可

3. **降阶建模章节缺少对PINN-ROM的完整论证**

   - 变更日志显示这是v2从骨架扩写的内容，扩写深度可能不足
   - **修复建议**：补充一个具体的PINN-ROM精度对比数值案例，或至少给出"适用条件"（数据充分、物理约束边界清晰）

## P2 改进建议（锦上添花）

1. **Muskingum-IDZ对偶性证明（§4.3.4）逻辑细节**
   - 式(4-33)MacLaurin展开取一阶近似：H(s) ≈ 1 - Ks + K²X(1-X)s²
   - 但实际上保留到一阶是：H(s) ≈ 1 - Ks + K(1-X)·KXs² - ...  ≈ 1 + (τ_m - τ_d)s
   - 式(4-34)的展开结果"1 + (τ_m - τ_d)s + ..."应与式(4-33)的"1 - Ks + ..."进行更严格的参数映射
   - 对偶关系K↔τ_d的结论正确，但推导过程中一阶项-Ks对应-(τ_d)s，而+τ_ms项来自分子，这一细节值得更清晰展示

2. **式(4-4)降阶矩阵的数学精度**
   - A_rom = V^T · A · V（Galerkin投影）
   - 文中未区分Galerkin投影和Petrov-Galerkin投影
   - 对于研究生教材，应说明当V不正交时需要左投影矩阵W（W^T · A · V），正交V的特殊情况W=V
   - 建议加一行说明"当V为正交矩阵时，V^T = V^{-1}，降阶矩阵具有最优的Frobenius范数误差"

3. **频域辨识的三维门槛（§4.5.4）缺乏工程来源**
   - 三维门槛：幅值误差<10%，相位误差<5°，约束违例率<2%
   - 这些数字是否有文献依据，还是经验规则？建议标注来源或说明"工程经验值"

4. **降阶模型上线检查清单（10项）可与章节小节编号对应**
   - 当前清单10项列出了与正文各节对应的检查内容，但未标注对应的小节编号
   - 建议每项后加"（见§4.X.X）"，方便读者查阅

5. **图表占位符**
   - 本章缺少：Muskingum入流-出流对比图、IDZ参数辨识Bode图示例、双轨切换策略流程图
